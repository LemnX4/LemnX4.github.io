Équation de Pauli

L'équation de Pauli est une équation de champs quantiques qui décrit le comportement des fermions lors d'une limite non-relativiste.

Définition​

L'équation de Pauli générale est la suivante :

iℏ∂Φ(t,x⃗)∂t=[(σ⃗⋅π⃗)22m+qV]Φ(t,x⃗)i\hbar \frac{\partial\Phi(t, \vec{x})}{\partial t} = \left [ \frac{\left ( \vec{\sigma}\cdot\vec{\pi}\right )^2}{2m} + qV \right ]\Phi(t, \vec{x})iℏ∂t∂Φ(t,x)​=[2m(σ⋅π)2​+qV]Φ(t,x)

Avec π⃗≡p⃗−qA⃗=iℏ∇⃗−qA⃗\vec{\pi} \equiv \vec{p} - q\vec{A} = i\hbar\vec{\nabla} - q\vec{A}π≡p​−qA=iℏ∇−qA où p⃗\vec{p}p​ est la quantité de mouvement, A⃗\vec{A}A le potentiel vecteur électromagnétique, VVV le potentiel électrique et qqq la charge du système étudié. Ici Φ\PhiΦ le premier spineur de Dirac du bi-spineur de Dirac.

Lorsqu'on explicite l'expression du produit σ⃗⋅π⃗\vec{\sigma}\cdot\vec{\pi}σ⋅π grâce à la relation de Pauli on obtient l'équation de Pauli standard :

iℏ∂Φ(t,x⃗)∂t=[∥p⃗−qA⃗∥22m−ℏq2mσ⃗⋅B⃗+qV]Φ(t,x⃗)i\hbar \frac{\partial\Phi(t, \vec{x})}{\partial t} = \left [ \frac{\left \| \vec{p} - q\vec{A} \right \|^2}{2m} - \frac{\hbar q}{2m}\vec{\sigma}\cdot\vec{B} + qV \right ]\Phi(t, \vec{x})iℏ∂t∂Φ(t,x)​=​2m​p​−qA​2​−2mℏq​σ⋅B+qV​Φ(t,x)

Même alors que la limite relativiste est appliquée à l'équation de Dirac relativiste on constate que l'équation qui en résulte porte sur un spineur et non pas un scalaire. La propriété intrinsèquement relativiste des fermions est mathématiquement structurelle et inscrite dans la description spinorielle de ceux-ci.

Construction par l'équation de Dirac​

Lorsqu'on applique un champ électromagnétique à notre système les nouvelles équations de Dirac pseudo-stationnaires sont les suivantes :

iℏ∂Φ∂t=cσ⃗⋅π⃗X+qVΦi\hbar \frac{\partial\Phi}{\partial t} = c\vec{\sigma}\cdot\vec{\pi}\mathcal{X} + qV\Phiiℏ∂t∂Φ​=cσ⋅πX+qVΦ iℏ∂X∂t=cσ⃗⋅π⃗Φ+(qV−2mc2)Xi\hbar\frac{\partial\mathcal{X}}{\partial t} = c\vec{\sigma}\cdot\vec{\pi}\Phi + \left ( qV - 2mc^2 \right ) \mathcal{X}iℏ∂t∂X​=cσ⋅πΦ+(qV−2mc2)X

Dans le cas de la limite non-relativiste, on suppose que les variations temporelles et l'énergie électrostatique sont petites devant mc2mc^2mc2. Nous obtenons alors cette égalité lors de l'approximation :

X=σ⃗⋅π⃗2mcΦ\mathcal{X} = \frac{\vec{\sigma}\cdot\vec{\pi}}{2mc}\Phi X=2mcσ⋅π​Φ

On peut injecter ce résultat dans la première équation :

iℏ∂Φ∂t=[(σ⃗⋅π⃗)22m+qV]Φi\hbar \frac{\partial\Phi}{\partial t} = \left [ \frac{\left ( \vec{\sigma}\cdot\vec{\pi}\right )^2}{2m} + qV \right ]\Phiiℏ∂t∂Φ​=[2m(σ⋅π)2​+qV]Φ

Il s'agit de l'équation de Pauli sous sa forme la plus générale. Nous voyons une analogie direct avec l'équation de Schrödinger. L'hamiltonien serait alors le suivant :

H=(σ⃗⋅π⃗)22m+qV\mathcal{H} = \frac{\left ( \vec{\sigma}\cdot\vec{\pi}\right )^2}{2m} + qV H=2m(σ⋅π)2​+qV

Nous pouvons utiliser la relation de Pauli sur le vecteur π⃗\vec{\pi}π :

(σ⃗⋅π⃗)2=(π⃗)2I2+iσ⃗⋅(π⃗∧π⃗)\left ( \vec{\sigma}\cdot\vec{\pi}\right )^2 = \left ( \vec{\pi} \right )^2 I_2 + i\vec{\sigma}\cdot\left ( \vec{\pi}\land\vec{\pi} \right )(σ⋅π)2=(π)2I2​+iσ⋅(π∧π)

Explicitement :

[πj,πk]=[pj−qAj,pk−qAk]\left [ \pi^j, \pi^k\right ] = \left [ p^j -qA^j, p^k-qA^k\right ][πj,πk]=[pj−qAj,pk−qAk] =[pj,pk]−q[pj,Ak]−q[Aj,pk]+q2[Aj,Ak]=−q[pj,Ak]−q[Aj,pk]= \left [ p^j, p^k\right ] - q\left [ p^j , A^k\right ] - q\left [ A^j, p^k\right ] + q^2 \left [ A^j, A^k\right ] = - q\left [ p^j , A^k\right ] - q\left [ A^j, p^k\right ]=[pj,pk]−q[pj,Ak]−q[Aj,pk]+q2[Aj,Ak]=−q[pj,Ak]−q[Aj,pk]

Sachant que pi=−iℏ∂ip^i=-i\hbar\partial_ipi=−iℏ∂i​ :

[πj,πk]=iℏq([∂j,Ak]−[∂k,Aj])\left [ \pi^j, \pi^k\right ] = i\hbar q \left ( \left [ \partial_j , A^k\right ] - \left [\partial_k, A^j \right ] \right )[πj,πk]=iℏq([∂j​,Ak]−[∂k​,Aj])

Appliquons le terme entre parenthèse à une fonction quelconque :

([∂j,Ak]−[∂k,Aj])f=[∂j,Ak]f−[∂k,Aj]f\left ( \left [ \partial_j , A^k\right ] - \left [\partial_k, A^j \right ] \right )f = \left [ \partial_j , A^k\right ]f - \left [\partial_k, A^j \right ]f([∂j​,Ak]−[∂k​,Aj])f=[∂j​,Ak]f−[∂k​,Aj]f =∂j(Akf)−Ak∂jf−∂k(Ajf)−Aj∂kf= \partial_j (A^k f) - A^k\partial_j f - \partial_k (A^j f) - A^j\partial_k f=∂j​(Akf)−Ak∂j​f−∂k​(Ajf)−Aj∂k​f =f∂jAk+Ak∂jf−Ak∂jf−f∂kAj−Aj∂kf−Aj∂kf=(∂jAk−∂kAj)f= f \partial_j A^k + A^k \partial_j f - A^k\partial_j f - f\partial_k A^j - A^j\partial_k f - A^j\partial_k f = \left ( \partial_j A^k - \partial_k A^j\right )f=f∂j​Ak+Ak∂j​f−Ak∂j​f−f∂k​Aj−Aj∂k​f−Aj∂k​f=(∂j​Ak−∂k​Aj)f

On a alors cette égalité :

[πj,πk]=iℏq(∂jAk−∂kAj)\left [ \pi^j, \pi^k\right ] = i\hbar q \left ( \partial_j A^k - \partial_k A^j \right )[πj,πk]=iℏq(∂j​Ak−∂k​Aj)

Et donc :

(π⃗∧π⃗)i=ϵijk[πj,πk]=iℏqϵijk(∂jAk−∂kAj)=iℏq(∇⃗∧A⃗)i\left ( \vec{\pi}\land\vec{\pi} \right )_i = \epsilon_{ijk}\left [ \pi^j, \pi^k\right ] = i\hbar q \epsilon_{ijk}\left ( \partial_j A^k - \partial_k A^j \right ) = i\hbar q\left ( \vec{\nabla}\land\vec{A} \right )_i(π∧π)i​=ϵijk​[πj,πk]=iℏqϵijk​(∂j​Ak−∂k​Aj)=iℏq(∇∧A)i​

D'après la définition du potentiel vecteur nous avons B⃗=∇⃗∧A⃗\vec{B} = \vec{\nabla}\land\vec{A}B=∇∧A Et donc ce résultat :

(σ⃗⋅π⃗)2=(π⃗)2I2−ℏqσ⃗⋅B⃗\left ( \vec{\sigma}\cdot\vec{\pi}\right )^2 = \left ( \vec{\pi} \right )^2 I_2 -\hbar q \vec{\sigma}\cdot\vec{B}(σ⋅π)2=(π)2I2​−ℏqσ⋅B

L'hamiltonien devient alors le suivant :

H=∥p⃗−qA⃗∥22m−ℏq2mσ⃗⋅B⃗+qV\mathcal{H} = \frac{\left \| \vec{p} - q\vec{A} \right \|^2}{2m} - \frac{\hbar q}{2m}\vec{\sigma}\cdot\vec{B} + qV H=2m​p​−qA​2​−2mℏq​σ⋅B+qV

Nous obtenons l'équation de Pauli standard.
