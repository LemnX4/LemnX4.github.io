Courant conservé

Le courant conservé est une quantité qui se conserve lors d'une transformation infinitésimale qui laisse l'action invariante.

Énoncé​

Soit les transformations infinitésimales suivantes :

{x′μ=xμ+Cjμαjφi′=φi+Fijαj\left \{ \begin{aligned} x'^\mu &= x^\mu + C^\mu_j\alpha^j \\ \varphi'_i &= \varphi_i + F_{ij}\alpha^j \end{aligned} \right .{x′μφi′​​=xμ+Cjμ​αj=φi​+Fij​αj​

Où αj\alpha^jαj sont infinitésimales. Lorsque l'action est invariante sous ces transformations nous obtenons un courant conservé Φjμ\Phi^\mu_jΦjμ​ :

δS=0        ⟺        ∂μΦjμ=0\delta \mathscr{S} = 0 \ \ \ \ \ \ \iff \ \ \ \ \ \ \partial_\mu \Phi^\mu_j = 0δS=0      ⟺      ∂μ​Φjμ​=0

L'expression de ce courant conservé est la suivante :

Φjμ≡∂L∂(∂μφi)∂νφiCjν−∂L∂(∂μφi)Fij−LCjμ\Phi^\mu_j \equiv \frac{\partial \mathscr{L}}{\partial (\partial_\mu \varphi_i)}\partial_\nu \varphi_i C^\nu_j - \frac{\partial \mathscr{L}}{\partial (\partial_\mu \varphi_i)} F_{ij} - \mathscr{L}C^\mu_j Φjμ​≡∂(∂μ​φi​)∂L​∂ν​φi​Cjν​−∂(∂μ​φi​)∂L​Fij​−LCjμ​

Nous avons ces transformations infinitésimales :

{x′μ=xμ+δxμ=xμ+Cjμαjφi′=φi+δφi=φi+Fijαj\left \{ \begin{aligned} x'^\mu &= x^\mu + \delta x^\mu = x^\mu + C^\mu_j\alpha^j \\ \varphi'_i &= \varphi_i + \delta \varphi_i = \varphi_i + F_{ij}\alpha^j \end{aligned} \right .{x′μφi′​​=xμ+δxμ=xμ+Cjμ​αj=φi​+δφi​=φi​+Fij​αj​
