Équations d'Euler-Lagrange

Les équations d'Euler-Lagrange sont les équations du mouvement des champs.

Énoncé​

Lorsque l'action est stationnaire selon le principe de moindre action nous obtenons les équations d'Euler-Lagrange comme étant les équations du mouvement des champs :

δS=0        ⟺        ∂L∂φi−∂μ[∂L∂(∂μφi)]=0\delta \mathcal{S} = 0 \ \ \ \ \ \ \iff \ \ \ \ \ \ \frac{\partial \mathscr{L}}{\partial \varphi_i} - \partial_\mu \left [ \frac{\partial \mathscr{L}}{\partial (\partial_\mu \varphi_i)} \right ] = 0δS=0      ⟺      ∂φi​∂L​−∂μ​[∂(∂μ​φi​)∂L​]=0
