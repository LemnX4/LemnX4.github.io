Tenseur énergie-impulsion

Le tenseur énergie-impulsion est une quantité qui renseigne sur le contenus énergétique de l'espace-temps.

Définition​

Afin de décrire ce que contient l'espace-temps le tenseur qui convient est le tenseur énergie-impulsion TμνT^{\mu\nu}Tμν. Il répondra, par définition, à cette relation de conservation :

∇μTμν=0\nabla_\mu T^{\mu\nu} = 0∇μ​Tμν=0

Nous avons alors T00T^{00}T00 la densité volumique d'énergie, T0jT^{0j}T0j la densité surfacique de flux d'énergie, Ti0T^{i0}Ti0 la densité volumique de flux de quantité de mouvement. TijT^{ij}Tij la densité surfacique de flux de quantité de mouvement.

E=∭VT00dV=∬ST0jdSjE = \iiint_V T^{00} dV = \iint_S T^{0j} dS_jE=∭V​T00dV=∬S​T0jdSj​ mvi=∭VT0idV=∬STijdSjmv^i = \iiint_V T^{0i} dV = \iint_S T^{ij} dS_jmvi=∭V​T0idV=∬S​TijdSj​

L'expression de ce tenseur dépend de ce qu'on veut étudier. Si est l'espace-temps est vide, il n'y a pas de densité d'énergie et dans ce cas le tenseur énergie-impulsion est nul.

Tμν=(ρm+pc2)uμuν−pgμνT^{\mu\nu} = \left ( \rho_m + \frac{p}{c^2} \right ) u^\mu u^\nu - p g^{\mu\nu}Tμν=(ρm​+c2p​)uμuν−pgμν

T αα=(ρm+pc2)uαuα−pg αα=(ρm+pc2)c2−4pT^{\alpha}_{\ \alpha} = \left ( \rho_m + \frac{p}{c^2} \right ) u^\alpha u_\alpha - p g^{\alpha}_{\ \alpha} =\left ( \rho_m + \frac{p}{c^2} \right ) c^2 - 4 pT αα​=(ρm​+c2p​)uαuα​−pg αα​=(ρm​+c2p​)c2−4p
