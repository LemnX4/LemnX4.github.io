Constante cosmologique

La constante cosmologique est une origine mathématique de l'expansion de l'univers.

Justification mathématique​

La nullité de la quadri-divergence du tenseur métrique nous autorise a ajouter le tenseur métrique avec un facteur multiplicatif dans les équations :

Gμν±Λgμν=κTμνG_{\mu\nu} \pm \Lambda g_{\mu\nu} = \kappa T_{\mu\nu}Gμν​±Λgμν​=κTμν​

On peut réarranger cette équation pour exhiber le coté énergétique de cette constante :

Gμν=κTμν∓ΛgμνG_{\mu\nu} = \kappa T_{\mu\nu} \mp \Lambda g_{\mu\nu}Gμν​=κTμν​∓Λgμν​

Soit cette constante est relié intrinsèquement à un l'espace-temps, soit à son contenu.

Effets physiques​

Historiquement, Einstein introduisis cette constante avec un signe négatif, avec comme idée que l'Univers contienne plus d'énergie via un champ intrinsèque, une énergie du vide.

En effet, si on pose cette constante positive ou négative, on obtient cette équation :

Gμν=κ(Tμν∓Λκgμν)=κ(Tμν+TμνΛ)G_{\mu\nu} = \kappa \left ( T_{\mu\nu} \mp \frac{\Lambda}{\kappa} g_{\mu\nu} \right ) = \kappa \left ( T_{\mu\nu} + T^{\Lambda}_{\mu\nu} \right ) Gμν​=κ(Tμν​∓κΛ​gμν​)=κ(Tμν​+TμνΛ​)

Avec l'introduction d'une nouvelle quantité, le tenseur énergie-impulsion cosmologique :

TμνΛ≡∓Λc48πGgμνT^{\Lambda}_{\mu\nu} \equiv \mp\frac{\Lambda c^4}{8\pi G} g_{\mu\nu}TμνΛ​≡∓8πGΛc4​gμν​

Si on impose que le fluide correspondant à la constante cosmologique est parfait, d'après l'expression du tenseur énergie-impulsion associé à ce cas particulier, nous obtenons ces équations :

(ρm+pc2)uμuν−pgμν=∓Λc48πGgμν\left ( \rho_m + \frac{p}{c^2} \right ) u_\mu u_\nu - p g_{\mu\nu} = \mp\frac{\Lambda c^4}{8\pi G} g_{\mu\nu}(ρm​+c2p​)uμ​uν​−pgμν​=∓8πGΛc4​gμν​

Une équation en résulte immédiatement pour la densité d'énergie :

ρ=−p\rho = -pρ=−p

De même pour la pression :

p=±Λc48πGp = \pm \frac{\Lambda c^4}{8\pi G}p=±8πGΛc4​

Ce fluide est particulier : si sa densité d'énergie est positive, sa pression est négative et inversement.

Les observations cosmologiques indiquent que cette constante cosmologique possède une pression positive. Il s'agirait d'une énergie à caractère négative. Pour que la pression soit positive il faut que le signe devant le constante cosmologique soit positif.

L'introduction de cette constante change la limite newtonienne de ces équations. Reprenons les indices temporelles :

R00=κ(T00−12g00T)−Λg00R_{00} = \kappa \left ( T_{00} - \frac{1}{2} g_{00} T \right ) - \Lambda g_{00}R00​=κ(T00​−21​g00​T)−Λg00​ ΔΦGc2=κρmc22−Λ\frac{\Delta \Phi_G}{c^2} = \kappa \frac{\rho_m c^2}{2} - \Lambdac2ΔΦG​​=κ2ρm​c2​−Λ ΔΦG=8πGc4ρmc42−Λc2\Delta \Phi_G =\frac{8\pi G}{c^4}\frac{\rho_m c^4}{2} - \Lambda c^2ΔΦG​=c48πG​2ρm​c4​−Λc2

Ce qui nous donne la nouvelle équation de Poisson avec la constante cosmologique :

ΔΦG=4πGρm−Λc2\Delta \Phi_G = 4\pi G \rho_m - \Lambda c^2ΔΦG​=4πGρm​−Λc2

Ce qui donne le résultat suivant :

−∇⃗ΦG=−GMr2u⃗r+Λc2r3u⃗r-\vec{\nabla} \Phi_G = -\frac{GM}{r^2}\vec{u}_r + \frac{\Lambda c^2r}{3}\vec{u}_r−∇ΦG​=−r2GM​ur​+3Λc2r​ur​

La constante cosmologique a un effet d'expansion.
