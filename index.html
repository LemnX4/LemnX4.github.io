Conventions sur les indices

Convention de sommation d'Einstein :

Lorsqu'un indice est répété deux fois dans un produit de termes, il s'agit d'une somme sur toutes les valeurs possibles de cet indice. Ces indices répétés sont dit muets.

Les indices latins prennent les valeurs 1, 2 et 3.

AiBi=∑i=1i=3AiBi=A1B1+A2B2+A3B3A_i B^i = \overset{i=3}{\underset{i=1}{\sum}} A_i B^i = A_1 B^1 + A_2 B^2 + A_3 B^3Ai​Bi=i=1∑​i=3​Ai​Bi=A1​B1+A2​B2+A3​B3

Les indices grecs prennent les valeurs 0, 1, 2 et 3.

AμBμ=∑μ=0μ=3AμBμ=A0B0+A1B1+A2B2+A3B3A_\mu B^\mu = \overset{\mu=3}{\underset{\mu=0}{\sum}} A_\mu B^\mu = A_0 B^0 + A_1 B^1 + A_2 B^2 + A_3 B^3Aμ​Bμ=μ=0∑​μ=3​Aμ​Bμ=A0​B0+A1​B1+A2​B2+A3​B3

La signature de la métrique utilisée est (+,−,−,−)(+,-,-,-)(+,−,−,−).
