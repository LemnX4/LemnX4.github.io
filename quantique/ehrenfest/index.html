Théorème d'Ehrenfest

Définition​

La moyenne de la variation temporelle d'un opérateur est reliée à la variation temporelle de sa moyenne par le théorème d'Ehrenfest :

∂∂t⟨A^⟩=iℏ⟨[H^,A^]⟩+⟨∂A^∂t⟩\frac{\partial}{\partial t}\langle\hat{A}\rangle = \frac{i}{\hbar}\left \langle \left [ \hat{\mathcal{H}},\hat{A} \right ] \right \rangle + \left \langle \frac{\partial \hat{A}}{\partial t} \right \rangle∂t∂​⟨A^⟩=ℏi​⟨[H^,A^]⟩+⟨∂t∂A^​⟩

Où H^\hat{\mathcal{H}}H^ est l'opérateur hamiltonien.

Construction avec l'équation de Schrödinger​

Détrminons quelle est l'évolution temporelle de la moyenne d'une observable :

∂∂t⟨A^⟩=∂∂t(⟨Ψ∣A^∣Ψ⟩)\frac{\partial}{\partial t}\langle\hat{A}\rangle = \frac{\partial}{\partial t}\left ( \langle\Psi|\hat{A}|\Psi\rangle \right )∂t∂​⟨A^⟩=∂t∂​(⟨Ψ∣A^∣Ψ⟩)

On peut développer :

∂∂t(⟨Ψ∣A^∣Ψ⟩)=(ddt⟨Ψ∣)A^∣Ψ⟩+⟨Ψ∣∂A^∂t∣Ψ⟩+⟨Ψ∣A^(ddt∣Ψ⟩)\frac{\partial}{\partial t}\left ( \langle\Psi|\hat{A}|\Psi\rangle \right ) = \left (\frac{d}{dt}\langle\Psi|\right )\hat{A}|\Psi\rangle + \langle\Psi|\frac{\partial \hat{A}}{\partial t}|\Psi\rangle + \langle\Psi|\hat{A}\left (\frac{d}{dt}|\Psi\rangle\right )∂t∂​(⟨Ψ∣A^∣Ψ⟩)=(dtd​⟨Ψ∣)A^∣Ψ⟩+⟨Ψ∣∂t∂A^​∣Ψ⟩+⟨Ψ∣A^(dtd​∣Ψ⟩)

Nous connaissons l'expression des dérivées temporelles d'états grâce à l'équation de Schrödinger :

ddt∣Ψ⟩=−iℏH^∣Ψ⟩\frac{d}{dt}| \Psi \rangle = - \frac{i}{\hbar}\hat{\mathcal{H}}| \Psi \rangledtd​∣Ψ⟩=−ℏi​H^∣Ψ⟩ ddt⟨Ψ∣=iℏ⟨Ψ∣H^\frac{d}{dt}\langle \Psi | = \frac{i}{\hbar}\langle \Psi | \hat{\mathcal{H}}dtd​⟨Ψ∣=ℏi​⟨Ψ∣H^

L'expression est alors la suivante :

∂∂t(⟨Ψ∣A^∣Ψ⟩)=iℏ⟨Ψ∣H^A^∣Ψ⟩+⟨Ψ∣∂A^∂t∣Ψ⟩−iℏ⟨Ψ∣A^H^∣Ψ⟩\frac{\partial}{\partial t}\left ( \langle\Psi|\hat{A}|\Psi\rangle \right ) = \frac{i}{\hbar}\langle \Psi | \hat{\mathcal{H}}\hat{A}|\Psi\rangle + \langle\Psi|\frac{\partial \hat{A}}{\partial t}|\Psi\rangle - \frac{i}{\hbar}\langle\Psi|\hat{A}\hat{\mathcal{H}}| \Psi \rangle∂t∂​(⟨Ψ∣A^∣Ψ⟩)=ℏi​⟨Ψ∣H^A^∣Ψ⟩+⟨Ψ∣∂t∂A^​∣Ψ⟩−ℏi​⟨Ψ∣A^H^∣Ψ⟩

On peut regrouper des termes :

∂∂t(⟨Ψ∣A^∣Ψ⟩)=iℏ⟨Ψ∣(H^A^−A^H^)∣Ψ⟩+⟨Ψ∣∂A^∂t∣Ψ⟩\frac{\partial}{\partial t}\left ( \langle\Psi|\hat{A}|\Psi\rangle \right ) = \frac{i}{\hbar}\langle \Psi | \left ( \hat{\mathcal{H}}\hat{A} - \hat{A}\hat{\mathcal{H}}\right ) |\Psi\rangle + \langle\Psi|\frac{\partial \hat{A}}{\partial t}|\Psi\rangle∂t∂​(⟨Ψ∣A^∣Ψ⟩)=ℏi​⟨Ψ∣(H^A^−A^H^)∣Ψ⟩+⟨Ψ∣∂t∂A^​∣Ψ⟩

On peut alors voir un communtateur :

[H^,A^]=H^A^−A^H^\left [ \hat{\mathcal{H}},\hat{A} \right ] = \hat{\mathcal{H}}\hat{A} - \hat{A}\hat{\mathcal{H}}[H^,A^]=H^A^−A^H^

Nous retrouvons bien le théorème d'Ehrenfest.
