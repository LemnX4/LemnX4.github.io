Équation de Schrödinger

Définition​

L'équation fondamentale de la physique quantique a été postulée par le physicien Erwin Schrödinger. L'équation de Schrödinger non relativiste la plus générale est la suivante :

H^∣Ψ(t)⟩=iℏddt∣Ψ(t)⟩\hat{\mathcal{H}}| \Psi(t) \rangle = i\hbar\frac{d}{dt}| \Psi(t) \rangleH^∣Ψ(t)⟩=iℏdtd​∣Ψ(t)⟩

Si elle s'applique aux états quantiques, elle s'applique aussi aux fonctions d'onde. Connaissant l'expression de l'opérateur hamiltonien en représentation différentielle, nous avons alors l'équation de Schrödinger pour les fonctions d'onde bosoniques :

−ℏ22mΔΨ(r⃗,t)+V(t,r⃗)Ψ(t,r⃗)=iℏ∂∂tΨ(r⃗,t)-\frac{\hbar^2}{2m}\Delta\Psi(\vec{r},t) + V(t,\vec{r})\Psi(t,\vec{r}) = i\hbar\frac{\partial}{\partial t}\Psi(\vec{r},t)−2mℏ2​ΔΨ(r,t)+V(t,r)Ψ(t,r)=iℏ∂t∂​Ψ(r,t)

Cette équation provient de la limite non-relativiste de l'équation de Klein-Gordon auquel un potentiel quelconque a été éjouté. Si l'état quantique ne dépend pas du temps, l'équation prend alors sa forme indépendante du temps :

H^∣Ψ⟩=E∣Ψ⟩\hat{\mathcal{H}}| \Psi \rangle = E | \Psi \rangleH^∣Ψ⟩=E∣Ψ⟩

Où l'opérateur hamiltonien ne dépend pas du temps implicitement. De même pour les fonctions d'onde :

−ℏ22mΔΨ(r⃗)+V(r⃗)Ψ(r⃗)=EΨ(r⃗)-\frac{\hbar^2}{2m}\Delta\Psi(\vec{r}) +V(\vec{r})\Psi(\vec{r}) = E\Psi(\vec{r})−2mℏ2​ΔΨ(r)+V(r)Ψ(r)=EΨ(r)

On peut voir que si ∣Ψ1(t)⟩| \Psi_1(t) \rangle∣Ψ1​(t)⟩ et ∣Ψ2(t)⟩| \Psi_2(t) \rangle∣Ψ2​(t)⟩ sont solutions de l'équation de Schrödinger, alors la composition linéaire de ces deux solutions est aussi une solution. Nous avons ici l'appliquation directe d'une décomposition selon une base. En effet, puisque les vecteurs d'état peuvent se décomposer dans une base quelconque, on a cette même propriété pour les états quantiques :

∣Ψ(t)⟩=∑nλn∣Ψn(t)⟩| \Psi(t) \rangle = \sum_n \lambda_n | \Psi_n(t) \rangle∣Ψ(t)⟩=n∑​λn​∣Ψn​(t)⟩

Où les états ∣Ψk(t)⟩| \Psi_k(t) \rangle∣Ψk​(t)⟩ sont donc orthogonaux pour constituer une base. Nous avons donc aussi une condition de renormalisation pour que le caractère probabiliste soit respecté :

∑k∣λk∣2=1\sum_k |\lambda_k|^2 = 1k∑​∣λk​∣2=1

Cette condition respecte bien le principe de superposition quantique. La densité de probabilité ρ\rhoρ et le courant de probabilité j⃗\vec{j}j​ on cette expression :

ρ=Ψ∗Ψ\rho = \Psi^*\Psiρ=Ψ∗Ψ j⃗=−iℏ2m(Ψ∗∇⃗Ψ−Ψ∇⃗Ψ∗)\vec{j} = -i\frac{\hbar}{2m}\left ( \Psi^*\vec{\nabla}\Psi- \Psi\vec{\nabla}\Psi^*\right )j​=−i2mℏ​(Ψ∗∇Ψ−Ψ∇Ψ∗)

Ils répondent à l'équation de conservation. Cette équation de conservation ne fait intervenir aucuns postulats autres que celle de la validité de l'équation de Schrödinger : elle s'applique à toute la théorie quantique.

Évolution temporelle d'un état quantique​

L'équation de Schrödinger portant sur les états quantiques est une équation différentielle sur le temps :

ddt∣Ψ(t)⟩+iℏH^∣Ψ(t)⟩=ddt∣Ψ(t)⟩+iℏE∣Ψ(t)⟩=0\frac{d}{dt}| \Psi(t) \rangle + \frac{i}{\hbar}\hat{\mathcal{H}}| \Psi(t) \rangle = \frac{d}{dt}| \Psi(t) \rangle + \frac{i}{\hbar}E| \Psi(t) \rangle = 0dtd​∣Ψ(t)⟩+ℏi​H^∣Ψ(t)⟩=dtd​∣Ψ(t)⟩+ℏi​E∣Ψ(t)⟩=0

La solution de cette équation est la suivante :

∣Ψ(t)⟩=e−iℏEt∣Ψ(0)⟩| \Psi(t) \rangle = e^{-\frac{i}{\hbar}Et} | \Psi(0) \rangle∣Ψ(t)⟩=e−ℏi​Et∣Ψ(0)⟩

Prennons un état quantique ∣Ψ(t)⟩| \Psi(t) \rangle∣Ψ(t)⟩ tel que son état initial soit le suivant :

∣Ψ(0)⟩=∣u1⟩+∣u2⟩|\Psi(0)\rangle = |u_1\rangle + |u_2\rangle∣Ψ(0)⟩=∣u1​⟩+∣u2​⟩

Avec ∣u1⟩|u_1\rangle∣u1​⟩ et ∣u2⟩|u_2\rangle∣u2​⟩ des états de base. Ces états possèdent les énergies E1E_1E1​ et E2E_2E2​ respectivement. \ On va chercher à déterminer l'état pour n'importe quel temps.

∣Ψ(t)⟩=α(t)∣u1⟩+β(t)∣u2⟩|\Psi(t)\rangle = \alpha(t)|u_1\rangle + \beta(t)|u_2\rangle∣Ψ(t)⟩=α(t)∣u1​⟩+β(t)∣u2​⟩

On peut appliquer l'hamiltonien et utiliser l'équation de Schrödinger :

E1α(t)∣u1⟩+E2β(t)∣u2⟩=iℏdαdt∣u1⟩+iℏdβdt∣u2⟩E_1\alpha(t)|u_1\rangle + E_2\beta(t)|u_2\rangle = i\hbar\frac{d\alpha}{dt}|u_1\rangle + i\hbar\frac{d\beta}{dt}|u_2\rangleE1​α(t)∣u1​⟩+E2​β(t)∣u2​⟩=iℏdtdα​∣u1​⟩+iℏdtdβ​∣u2​⟩

On a alors ces deux équations :

dαdt=−iE1ℏα(t)dβdt=−iE2ℏβ(t)\begin{aligned} \frac{d\alpha}{dt} &= -i\frac{E_1}{\hbar}\alpha(t) \\ \frac{d\beta}{dt} &= -i\frac{E_2}{\hbar}\beta(t) \end{aligned}dtdα​dtdβ​​=−iℏE1​​α(t)=−iℏE2​​β(t)​

On peut résoudre ces équations différentielles :

α(t)=e−iE1ℏtβ(t)=e−iE2ℏt\begin{aligned} \alpha(t) &= e^{-i\frac{E_1}{\hbar}t} \\ \beta(t) &= e^{-i\frac{E_2}{\hbar}t} \end{aligned}α(t)β(t)​=e−iℏE1​​t=e−iℏE2​​t​

L'état est alors connu pour n'importe quel valeur temporelle :

∣Ψ(t)⟩=e−iE1ℏt∣u1⟩+e−iE2ℏt∣u2⟩|\Psi(t)\rangle = e^{-i\frac{E_1}{\hbar}t}|u_1\rangle + e^{-i\frac{E_2}{\hbar}t}|u_2\rangle∣Ψ(t)⟩=e−iℏE1​​t∣u1​⟩+e−iℏE2​​t∣u2​⟩

Séparation de variables spatio-temporelles pour un potentiel statique​

Nous allons décomposer la fonction d'onde en faisant une séparation de variable :

Ψ(r⃗,t)=f(r⃗)g(t)\Psi(\vec{r},t) = f(\vec{r})g(t)Ψ(r,t)=f(r)g(t)

On rappelle l'équation de Schrödinger unidimensionnelle :

−ℏ22mΔΨ(r⃗,t)+V(r⃗)Ψ(r⃗,t)=iℏ∂∂tΨ(r⃗,t)=EΨ(r⃗,t)-\frac{\hbar^2}{2m}\Delta \Psi(\vec{r},t) +V(\vec{r})\Psi(\vec{r},t) = i\hbar\frac{\partial}{\partial t}\Psi(\vec{r},t) = E\Psi(\vec{r},t)−2mℏ2​ΔΨ(r,t)+V(r)Ψ(r,t)=iℏ∂t∂​Ψ(r,t)=EΨ(r,t)

Nous pouvons faire la séparation de variable :

−ℏ22mΔ(f(r⃗)g(t))+V(r⃗)f(r⃗)g(t)=iℏ∂∂t(f(r⃗)g(t))=Ef(r⃗)g(t)-\frac{\hbar^2}{2m}\Delta \left ( f(\vec{r})g(t) \right ) +V(\vec{r})f(\vec{r})g(t)= i\hbar\frac{\partial}{\partial t}\left ( f(\vec{r})g(t) \right ) = Ef(\vec{r})g(t) −2mℏ2​Δ(f(r)g(t))+V(r)f(r)g(t)=iℏ∂t∂​(f(r)g(t))=Ef(r)g(t)

Ce qui nous donne :

(−ℏ22mΔf(r⃗)+V(x)f(r⃗))g(t)=f(r⃗)(iℏdg(t)dt)=Ef(r⃗)g(t)\left ( -\frac{\hbar^2}{2m}\Delta f(\vec{r}) +V(x)f(\vec{r}) \right )g(t)= f(\vec{r})\left ( i\hbar\frac{dg(t)}{d t}\right ) = Ef(\vec{r}) g(t)(−2mℏ2​Δf(r)+V(x)f(r))g(t)=f(r)(iℏdtdg(t)​)=Ef(r)g(t)

En faisant cette séparation, nous obtenons cette équation si on divise tout par Ψ(r⃗,t)=f(r⃗)g(t)\Psi(\vec{r},t)=f(\vec{r})g(t)Ψ(r,t)=f(r)g(t) :

1f(r⃗)(−ℏ22mΔf(r⃗)+V(r⃗)f(r⃗))=1g(t)(iℏdg(t)dt)=E\frac{1}{f(\vec{r})}\left ( -\frac{\hbar^2}{2m}\Delta f(\vec{r}) +V(\vec{r})f(\vec{r}) \right )= \frac{1}{g(t)}\left ( i\hbar\frac{dg(t)}{d t}\right ) = Ef(r)1​(−2mℏ2​Δf(r)+V(r)f(r))=g(t)1​(iℏdtdg(t)​)=E

Nous obtenons alors deux équations complétement indépendantes :

−ℏ22mΔf(r⃗)+V(x)f(r⃗)=Ef(r⃗)-\frac{\hbar^2}{2m}\Delta f(\vec{r}) +V(x)f(\vec{r}) = Ef(\vec{r})−2mℏ2​Δf(r)+V(x)f(r)=Ef(r) iℏdg(t)dt=Eg(t)i\hbar\frac{dg(t)}{d t}= Eg(t)iℏdtdg(t)​=Eg(t)

Nous voyons bien que la première est l'équation de Schrödinger indépendante du temps tandis que la deuxième est une application de l'opérateur énergie. \ On peut résoudre assez facilement la deuxième équation pour tout les potentiels possible puisque cette équation ne dépend pas du potentiel :

g(t)=g0e−iEℏtg(t) = g_0 e^{-i\frac{E}{\hbar}t}g(t)=g0​e−iℏE​t

La solution est donc de cette forme :

Ψ(r⃗,t)=g0f(r⃗)e−iEℏt\Psi(\vec{r},t) = g_0 f(\vec{r}) e^{-i\frac{E}{\hbar}t}Ψ(r,t)=g0​f(r)e−iℏE​t

L'équation qui va dépendre du potentiel est la première, c'est la fonction f(x)f(x)f(x) qui va varier en fonction du problème qu'on pose. \ La résolution de l'équation de Schrödinger pour un potentiel statique a pour but de déterminer quel est l'équation qui porte sur f(x)f(x)f(x) et quel est l'expression de cette fonction pour certains cas physiques particuliers.

Relation de dispersion pour une particule libre​

Pour une particule libre, le potentiel est nul par définition. L'équation de Schödinger est alors la suivante pour les cas unidimensionnels :

−ℏ22m∂2∂x2Ψ(x,t)=iℏ∂∂tΨ(x,t)-\frac{\hbar^2}{2m}\frac{\partial^2}{\partial x^2}\Psi(x,t) = i\hbar\frac{\partial}{\partial t}\Psi(x,t)−2mℏ2​∂x2∂2​Ψ(x,t)=iℏ∂t∂​Ψ(x,t)

Prennons la solution plane de l'équation de Schrödinger unidimensionnelle :

Ψ(x,t)=Ψ0ei(kx−ωt)\Psi(x,t) = \Psi_0e^{i(kx-\omega t)}Ψ(x,t)=Ψ0​ei(kx−ωt)

Injectons cette solution dans l'équation sachant ces relations :

∂2∂x2Ψ(x,t)=−k2Ψ(x,t)∂∂tΨ(x,t)=−iωΨ(x,t)\begin{aligned} \frac{\partial^2}{\partial x^2}\Psi(x,t) &= -k^2\Psi(x,t) \\ \frac{\partial}{\partial t}\Psi(x,t) &= -i\omega\Psi(x,t) \end{aligned}∂x2∂2​Ψ(x,t)∂t∂​Ψ(x,t)​=−k2Ψ(x,t)=−iωΨ(x,t)​

Nous obtenons cette relation en injectant dans l'équation :

ℏ22mk2=ℏω\frac{\hbar^2}{2m}k^2 = \hbar\omega2mℏ2​k2=ℏω

On a alors la puslation en fonction du nombre d'onde en valeur absolue :

ω=ℏ2mk2\omega = \frac{\hbar}{2m}k^2ω=2mℏ​k2

On voit alors que la pulsation est fonction du nombre d'onde. Il s'agit d'une relation de dispersion.

Construction des quantités conservées​

Reprennons l'équation de Schrödinger pour les fonctions d'onde :

−ℏ22mΔΨ+V(t,r⃗)Ψ=iℏ∂Ψ∂t-\frac{\hbar^2}{2m}\Delta\Psi +V(t,\vec{r})\Psi = i\hbar\frac{\partial \Psi}{\partial t}−2mℏ2​ΔΨ+V(t,r)Ψ=iℏ∂t∂Ψ​

On peut prendre le conjugué complexe de cette équation :

−ℏ22mΔΨ∗+V(t,r⃗)Ψ∗=−iℏ∂Ψ∗∂t-\frac{\hbar^2}{2m}\Delta\Psi^* +V(t,\vec{r})\Psi^* = -i\hbar\frac{\partial \Psi^*}{\partial t}−2mℏ2​ΔΨ∗+V(t,r)Ψ∗=−iℏ∂t∂Ψ∗​

Nous allons multiplier la première par la fonction d'onde conjuguée et la deuxième par la fonction d'onde simplement :

−ℏ22mΨ∗ΔΨ+V(t,r⃗)Ψ∗Ψ=iℏΨ∗∂Ψ∂t−ℏ22mΨΔΨ∗+V(t,r⃗)ΨΨ∗=−iℏΨ∂Ψ∗∂t\begin{aligned} -\frac{\hbar^2}{2m}\Psi^*\Delta\Psi +V(t,\vec{r})\Psi^*\Psi &= i\hbar\Psi^*\frac{\partial \Psi}{\partial t} \\ -\frac{\hbar^2}{2m}\Psi\Delta\Psi^* +V(t,\vec{r})\Psi\Psi^* &= -i\hbar\Psi\frac{\partial \Psi^*}{\partial t} \end{aligned}−2mℏ2​Ψ∗ΔΨ+V(t,r)Ψ∗Ψ−2mℏ2​ΨΔΨ∗+V(t,r)ΨΨ∗​=iℏΨ∗∂t∂Ψ​=−iℏΨ∂t∂Ψ∗​​

On peut soustraire ces deux équations :

−ℏ22mΨ∗ΔΨ+ℏ22mΨΔΨ∗=iℏΨ∗∂Ψ∂t+iℏΨ∂Ψ∗∂t-\frac{\hbar^2}{2m}\Psi^*\Delta\Psi + \frac{\hbar^2}{2m}\Psi\Delta\Psi^* = i\hbar\Psi^*\frac{\partial \Psi}{\partial t} + i\hbar\Psi\frac{\partial \Psi^*}{\partial t}−2mℏ2​Ψ∗ΔΨ+2mℏ2​ΨΔΨ∗=iℏΨ∗∂t∂Ψ​+iℏΨ∂t∂Ψ∗​

On factorise les dérivées temporelles :

−ℏ2mΨ∗ΔΨ+ℏ2mΨΔΨ∗=i∂(Ψ∗Ψ)∂t-\frac{\hbar}{2m}\Psi^*\Delta\Psi + \frac{\hbar}{2m}\Psi\Delta\Psi^* = i\frac{\partial (\Psi^*\Psi)}{\partial t}−2mℏ​Ψ∗ΔΨ+2mℏ​ΨΔΨ∗=i∂t∂(Ψ∗Ψ)​

On peut isoler la dérivée temporelle :

∂(Ψ∗Ψ)∂t−iℏ2m(Ψ∗ΔΨ−ΨΔΨ∗)=0\frac{\partial (\Psi^*\Psi)}{\partial t} - i\frac{\hbar}{2m}\left ( \Psi^*\Delta\Psi - \Psi\Delta\Psi^* \right ) = 0∂t∂(Ψ∗Ψ)​−i2mℏ​(Ψ∗ΔΨ−ΨΔΨ∗)=0

Nous allons utiliser une propriété de la divergence :

∇⃗⋅(a∇⃗b)=∇⃗a⋅∇⃗b+aΔb\vec{\nabla}\cdot\left ( a\vec{\nabla}b\right ) = \vec{\nabla}a \cdot\vec{\nabla}b + a\Delta b∇⋅(a∇b)=∇a⋅∇b+aΔb

On peut isoler le dernier terme :

aΔb=∇⃗⋅(a∇⃗b)−∇⃗a⋅∇⃗ba\Delta b = \vec{\nabla}\cdot\left ( a\vec{\nabla}b\right ) - \vec{\nabla}a \cdot\vec{\nabla}baΔb=∇⋅(a∇b)−∇a⋅∇b

Il est alors possible d'appliquer ce résultat à ce que nous avons dans les parenthèses :

Ψ∗ΔΨ−ΨΔΨ∗=∇⃗⋅(Ψ∗∇⃗Ψ)−∇⃗Ψ∗⋅∇⃗Ψ−(∇⃗⋅(Ψ∇⃗Ψ∗)−∇⃗Ψ⋅∇⃗Ψ∗)=∇⃗⋅(Ψ∗∇⃗Ψ)−∇⃗⋅(Ψ∇⃗Ψ∗)+∇⃗Ψ⋅∇⃗Ψ∗−∇⃗Ψ∗⋅∇⃗Ψ=∇⃗⋅(Ψ∗∇⃗Ψ)−∇⃗⋅(Ψ∇⃗Ψ∗)=∇⃗⋅(Ψ∗∇⃗Ψ−Ψ∇⃗Ψ∗)\begin{split} \Psi^*\Delta\Psi - \Psi\Delta\Psi^* &= \vec{\nabla}\cdot\left ( \Psi^*\vec{\nabla}\Psi\right ) - \vec{\nabla}\Psi^* \cdot\vec{\nabla}\Psi - \left ( \vec{\nabla}\cdot\left ( \Psi\vec{\nabla}\Psi^*\right ) - \vec{\nabla}\Psi \cdot\vec{\nabla}\Psi^* \right ) \\ &= \vec{\nabla}\cdot\left ( \Psi^*\vec{\nabla}\Psi\right ) - \vec{\nabla}\cdot\left ( \Psi\vec{\nabla}\Psi^*\right ) + \vec{\nabla}\Psi \cdot\vec{\nabla}\Psi^* - \vec{\nabla}\Psi^* \cdot\vec{\nabla}\Psi \\ &= \vec{\nabla}\cdot\left ( \Psi^*\vec{\nabla}\Psi\right ) - \vec{\nabla}\cdot\left ( \Psi\vec{\nabla}\Psi^*\right ) \\ &= \vec{\nabla}\cdot\left ( \Psi^*\vec{\nabla}\Psi- \Psi\vec{\nabla}\Psi^*\right ) \end{split}Ψ∗ΔΨ−ΨΔΨ∗​=∇⋅(Ψ∗∇Ψ)−∇Ψ∗⋅∇Ψ−(∇⋅(Ψ∇Ψ∗)−∇Ψ⋅∇Ψ∗)=∇⋅(Ψ∗∇Ψ)−∇⋅(Ψ∇Ψ∗)+∇Ψ⋅∇Ψ∗−∇Ψ∗⋅∇Ψ=∇⋅(Ψ∗∇Ψ)−∇⋅(Ψ∇Ψ∗)=∇⋅(Ψ∗∇Ψ−Ψ∇Ψ∗)​

L'équation obtenue est donc la suivante :

∂(Ψ∗Ψ)∂t−iℏ2m∇⃗⋅(Ψ∗∇⃗Ψ−Ψ∇⃗Ψ∗)=0\frac{\partial (\Psi^*\Psi)}{\partial t} - i\frac{\hbar}{2m}\vec{\nabla}\cdot\left ( \Psi^*\vec{\nabla}\Psi- \Psi\vec{\nabla}\Psi^*\right ) = 0∂t∂(Ψ∗Ψ)​−i2mℏ​∇⋅(Ψ∗∇Ψ−Ψ∇Ψ∗)=0

Il s'agit d'une équation de conservation. En effet, elle est de cette forme :

∂ρ∂t+∇⃗⋅j⃗=0\frac{\partial \rho}{\partial t} + \vec{\nabla}\cdot\vec{j} = 0∂t∂ρ​+∇⋅j​=0
