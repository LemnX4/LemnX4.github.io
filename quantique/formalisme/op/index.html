Opérateurs

Définition​

Soit A^\hat{A}A^ un opérateur quelconque. Il est noté avec un accent circonflexe pour le différencié d'un nombre. Cet opérateur peut être matriciel ou différentiel, cela dépend de la représentation. Prenons par exemple ces opérateurs :

A^=(1320),         B^=3∂∂x\hat{A} = \begin{pmatrix} 1 & 3 \\ 2 & 0 \end{pmatrix}, \ \ \ \ \ \ \ \ \ \hat{B} = 3\frac{\partial}{\partial x}A^=(12​30​),         B^=3∂x∂​

Avec A^\hat{A}A^ un opérateur en représentation matricielle et B^\hat{B}B^ un opérateur en représentation fonctionnelle. Nous pouvons respectivement les faire agir sur un vecteur d'état ∣φ⟩| \varphi \rangle∣φ⟩ ou bien une fonction f(x)f(x)f(x) :

A^∣φ⟩=(1320)(13i)=(1+9i2)\hat{A}| \varphi \rangle = \begin{pmatrix} 1 & 3 \\ 2 & 0 \end{pmatrix} \begin{pmatrix} 1 \\ 3i \end{pmatrix} = \begin{pmatrix} 1+9i \\ 2 \end{pmatrix}A^∣φ⟩=(12​30​)(13i​)=(1+9i2​) B^f(x)=3∂∂x(x2+3x−2)=6x+9\hat{B}f(x) = 3\frac{\partial}{\partial x} \left ( x^2 + 3x - 2 \right ) = 6x + 9B^f(x)=3∂x∂​(x2+3x−2)=6x+9

Dans les deux cas nous obtenons un nouveau vecteur d'état ∣ϕ⟩| \phi \rangle∣ϕ⟩ et une nouvelle fonction g(x)g(x)g(x):

A^∣φ⟩=∣ϕ⟩,          B^f(x)=g(x)\hat{A}| \varphi \rangle = | \phi \rangle, \ \ \ \ \ \ \ \ \ \ \hat{B}f(x) = g(x)A^∣φ⟩=∣ϕ⟩,          B^f(x)=g(x)

On dit alors que ∣φ⟩| \varphi \rangle∣φ⟩ est un vecteur propre de l'opérateur A^\hat{A}A^ et que aaa est sa valeur propre, un nombre complexe, lorsque cette relation est vérifiée :

A^∣φ⟩=a∣φ⟩\hat{A} | \varphi \rangle = a | \varphi \rangleA^∣φ⟩=a∣φ⟩

Le conjugué hermitique A^†\hat{A}^\dagA^† d'un opérateur A^\hat{A}A^ est l'opérateur transconjugué de celui-ci. Un opérateur est dit hermitien lorsqu'il est invariant sous la transformation de transconjuguaisons. Si A^\hat{A}A^ est hermitien, alors A^†=A^\hat{A}^\dag=\hat{A}A^†=A^.

Un opérateur U^\hat{U}U^ est dit unitaire si U^†U^=I\hat{U}^\dag \hat{U} = \mathbb{I}U^†U^=I.

Commutateurs​

Si aaa et bbb sont deux quantités quelconques, le commutateur de aaa et bbb est définit de cette façon :

[a,b]≡ab−ba[a,b] \equiv ab - ba[a,b]≡ab−ba

Si ces deux quantités ne sont pas commutatives, il est possible que le commutateur ne soit pas nul. Un exemple simple est celui de la non commutativité des matrices. Il faut donc être prudent lorsqu'on manipule des objets qui ne commutent pas forcément. On peut exhiber une propriété du commutateur :

[b,a]=−[a,b][b,a] = -[a,b][b,a]=−[a,b]

On voit aussi que [a,a]=0[a,a]=0[a,a]=0 tout le temps.

Nous pouvons aussi expliciter un commutateur particulier pour s'exercer :

[a,bc]=abc−bca[a,bc] = abc - bca[a,bc]=abc−bca

On peut ajouter 0=bac−bac0 = bac - bac0=bac−bac :

[a,bc]=abc−bca+bac−bac=(abc−bac)+(bac−bca)[a,bc] = abc - bca + bac - bac = (abc- bac) + (bac- bca)[a,bc]=abc−bca+bac−bac=(abc−bac)+(bac−bca) =(ab−ba)c+b(ac−ca)= (ab- ba)c + b(ac- ca)=(ab−ba)c+b(ac−ca)

Nous obtenons de nouveaux commutateurs. On obtient alors la propriété suivante :

[a,bc]=[a,b]c+b[a,c][a,bc] = [a,b]c + b[a,c][a,bc]=[a,b]c+b[a,c]

Valeur moyenne​

Expression​

La valeur moyenne ⟨A^⟩\langle \hat{A} \rangle⟨A^⟩ des valeurs possibles des observables de l'opérateur A^\hat{A}A^ sur l'état ∣Ψ⟩| \Psi \rangle∣Ψ⟩ est donnée par cette expression :

⟨A^⟩=⟨Ψ∣A^∣Ψ⟩⟨Ψ∣Ψ⟩\langle \hat{A} \rangle = \frac{\langle \Psi | \hat{A}| \Psi \rangle}{\langle \Psi | \Psi \rangle}⟨A^⟩=⟨Ψ∣Ψ⟩⟨Ψ∣A^∣Ψ⟩​

On peut prendre l'exemple de la quantité de mouvement p⃗^=−iℏ∇⃗\hat{\vec{p}} = -i\hbar\vec{\nabla}p​^​=−iℏ∇ où on suppose la fonction d'onde normalisée :

⟨p⃗^⟩=−iℏ⟨Ψ∣∇⃗∣Ψ⟩\langle \hat{\vec{p}} \rangle = -i\hbar\langle \Psi | \vec{\nabla} | \Psi \rangle⟨p​^​⟩=−iℏ⟨Ψ∣∇∣Ψ⟩

Il s'agit alors du produit scalaire entre ∣Ψ⟩| \Psi \rangle∣Ψ⟩ et ∇⃗∣Ψ⟩\vec{\nabla} | \Psi \rangle∇∣Ψ⟩ :

⟨p⃗^⟩=−iℏ∭R3Ψ∗(t,r⃗)∇⃗Ψ(r⃗,t)d3r\langle \hat{\vec{p}} \rangle = -i\hbar\iiint_{\mathbb{R}^3} \Psi^*(t,\vec{r}) \vec{\nabla} \Psi(\vec{r},t) d^3r⟨p​^​⟩=−iℏ∭R3​Ψ∗(t,r)∇Ψ(r,t)d3r

⟨A^⟩≡∑nanP(an)\langle \hat{A} \rangle \equiv \sum_n a_n \mathbb{P}(a_n)⟨A^⟩≡n∑​an​P(an​)

On connaît la probabilité que la quantité mesurée soit ana_nan​ dans la base des vecteurs propres dans le cas général où il n'y pas eu normalisation de la fonction d'onde précedemment :

P(an)=∣⟨en∣Ψ⟩∣2⟨Ψ∣Ψ⟩\mathbb{P}(a_n) = \frac{|\langle e_n | \Psi \rangle|^2}{\langle \Psi | \Psi \rangle}P(an​)=⟨Ψ∣Ψ⟩∣⟨en​∣Ψ⟩∣2​

Il est possible d'expliciter le conjugué et donc aussi de faire commuter le produit scalaire :

P(an)=⟨Ψ∣en⟩⟨en∣Ψ⟩⟨Ψ∣Ψ⟩\mathbb{P}(a_n) = \frac{\langle \Psi | e_n \rangle\langle e_n | \Psi \rangle}{\langle \Psi | \Psi \rangle}P(an​)=⟨Ψ∣Ψ⟩⟨Ψ∣en​⟩⟨en​∣Ψ⟩​

La valeur moyenne peut alors s'écrire de cette façon : :

⟨A^⟩=∑nan⟨Ψ∣en⟩⟨en∣Ψ⟩⟨Ψ∣Ψ⟩=1⟨Ψ∣Ψ⟩⟨Ψ∣(∑nan∣en⟩⟨en∣)∣Ψ⟩\langle \hat{A} \rangle = \sum_n a_n \frac{\langle \Psi | e_n \rangle\langle e_n | \Psi \rangle}{\langle \Psi | \Psi \rangle} = \frac{1}{\langle \Psi | \Psi \rangle} \langle \Psi |\left ( \sum_n a_n | e_n \rangle\langle e_n | \right ) | \Psi \rangle⟨A^⟩=n∑​an​⟨Ψ∣Ψ⟩⟨Ψ∣en​⟩⟨en​∣Ψ⟩​=⟨Ψ∣Ψ⟩1​⟨Ψ∣(n∑​an​∣en​⟩⟨en​∣)∣Ψ⟩

Puisque ∣en⟩| e_n \rangle∣en​⟩ sont les vecteurs propres, il est possible d'écrire an∣en⟩=A^∣en⟩a_n| e_n \rangle = \hat{A}| e_n \ranglean​∣en​⟩=A^∣en​⟩ :

⟨Ψ∣(∑nan∣en⟩⟨en∣)∣Ψ⟩=⟨Ψ∣(∑nA^∣en⟩⟨en∣)∣Ψ⟩\langle \Psi |\left ( \sum_n a_n | e_n \rangle\langle e_n | \right ) | \Psi \rangle = \langle \Psi |\left ( \sum_n \hat{A} | e_n \rangle\langle e_n | \right ) | \Psi \rangle⟨Ψ∣(n∑​an​∣en​⟩⟨en​∣)∣Ψ⟩=⟨Ψ∣(n∑​A^∣en​⟩⟨en​∣)∣Ψ⟩

⟨A^⟩=1⟨Ψ∣Ψ⟩⟨Ψ∣(∑n∣en⟩⟨en∣)A^∣Ψ⟩\langle \hat{A} \rangle = \frac{1}{\langle \Psi | \Psi \rangle}\langle \Psi |\left ( \sum_n | e_n \rangle\langle e_n | \right ) \hat{A}| \Psi \rangle⟨A^⟩=⟨Ψ∣Ψ⟩1​⟨Ψ∣(n∑​∣en​⟩⟨en​∣)A^∣Ψ⟩
