Éxperience de Stern et Gerlach​

Opérateur de spin​

On définit l'opérateur de spin S⃗^\hat{\vec{S}}S^ :

S⃗^≡ℏ2σ⃗\hat{\vec{S}} \equiv \frac{\hbar}{2}\vec{\sigma}S^≡2ℏ​σ

Les matrices de Pauli sont anticommutatives :

[σj,σk]=2iϵjklσl\left [ \sigma^j, \sigma^k \right ] = 2i\epsilon_{jkl}\sigma^l[σj,σk]=2iϵjkl​σl

Alors on a cette relation:

[ℏ2σj,ℏ2σk]=iℏϵjkl(ℏ2σl)\left [ \frac{\hbar}{2}\sigma^j, \frac{\hbar}{2}\sigma^k \right ] = i\hbar \epsilon_{jkl} \left ( \frac{\hbar}{2}\sigma^l \right )[2ℏ​σj,2ℏ​σk]=iℏϵjkl​(2ℏ​σl)

Et donc cette relation d'anticommutativité de l'opérateur de spin :

[S^j,S^k]=iℏϵjklS^l\left [ \hat{S}^j, \hat{S}^k \right ] = i\hbar \epsilon_{jkl}\hat{S}^l[S^j,S^k]=iℏϵjkl​S^l

D'après le principe d'incertitude d'Heisenberg nous avons donc une incertitude intrinsèque sur l'état de spin d'un système.

Moment cinétique intrinsèque​

L'opérateur moment cinétique L⃗^\hat{\vec{L}}L^ possède aussi des propriétés similaires à l'opérateur de spin S⃗^\hat{\vec{S}}S^ :

[L^j,L^k]=iℏϵjklL^l\left [ \hat{L}^j, \hat{L}^k \right ] = i\hbar \epsilon_{jkl}\hat{L}^l[L^j,L^k]=iℏϵjkl​L^l

Hors ici la présence du spin n'est pas liée à une rotation mais uniquement au caractère quantique et relativiste du système étudié avant d'en faire la limite non-relativiste. On dit alors que le spin est analogue à un moment cinétique intrinsèque.

Avec un champ magnétique​

En présence d'un champ magnétique uniforme l'hamiltonien de Pauli contient ce terme de spin :

H^spin=−qℏ2mσ⃗⋅B⃗=−qmS⃗^⋅B⃗\hat{\mathcal{H}}_\text{spin} = -\frac{q\hbar}{2m}\vec{\sigma}\cdot\vec{B} = -\frac{q}{m}\hat{\vec{S}}\cdot\vec{B}H^spin​=−2mqℏ​σ⋅B=−mq​S^⋅B

Pour un champ magnétique selon l'axe OzOzOz :

S^z=ℏ2σ3=(ℏ200ℏ2)\hat{S}^z = \frac{\hbar}{2}\sigma^3 = \begin{pmatrix} \frac{\hbar}{2} & 0 \\ 0 & \frac{\hbar}{2} \end{pmatrix}S^z=2ℏ​σ3=(2ℏ​0​02ℏ​​)

Les valeurs propres et états propres de la matrice σ3\sigma^3σ3 sont λ1,2=±1\lambda_{1, 2} = \pm 1λ1,2​=±1 et ∣Ψ1⟩=(10)|\Psi_1\rangle = \begin{pmatrix}1\\0\end{pmatrix}∣Ψ1​⟩=(10​), ∣Ψ2⟩=(01)|\Psi_2\rangle = \begin{pmatrix}0\\1\end{pmatrix}∣Ψ2​⟩=(01​) :

S^z∣Ψ1⟩=−ℏ2∣Ψ1⟩,              S^z∣Ψ2⟩=+ℏ2∣Ψ2⟩\hat{S}^z|\Psi_1\rangle = - \frac{\hbar}{2}|\Psi_1\rangle, \ \ \ \ \ \ \ \ \ \ \ \ \ \ \hat{S}^z|\Psi_2\rangle = + \frac{\hbar}{2}|\Psi_2\rangleS^z∣Ψ1​⟩=−2ℏ​∣Ψ1​⟩,              S^z∣Ψ2​⟩=+2ℏ​∣Ψ2​⟩

La solution peut alors s'exprimer dans cette base :

∣Ψz⟩=α∣Ψ1⟩+β∣Ψ2⟩=(αβ)|\Psi_z\rangle = \alpha |\Psi_1\rangle + \beta |\Psi_2\rangle = \begin{pmatrix} \alpha \\ \beta \end{pmatrix}∣Ψz​⟩=α∣Ψ1​⟩+β∣Ψ2​⟩=(αβ​)

Nous avons alors deux états possibles : l'état "haut" ∣↑⟩=∣Ψ1⟩|\uparrow\rangle=|\Psi_1\rangle∣↑⟩=∣Ψ1​⟩ et l'état "bas" ∣↓⟩=∣Ψ2⟩|\downarrow\rangle=|\Psi_2\rangle∣↓⟩=∣Ψ2​⟩ et donc l'état ∣Ψ⟩|\Psi\rangle∣Ψ⟩ est une combinaisons linéaire de ces états.

∣Ψz⟩=α∣↑⟩+β∣↓⟩|\Psi_z\rangle = \alpha |\uparrow\rangle + \beta |\downarrow\rangle∣Ψz​⟩=α∣↑⟩+β∣↓⟩

Avec ∣α∣2+∣β∣2=1|\alpha|^2+|\beta|^2 = 1∣α∣2+∣β∣2=1 pour assurer la normalisation de l'état. Pour obtenir l'énergie moyenne il faudra alors faire cette opération :

⟨S⃗⟩=⟨Ψ∣S⃗^∣Ψ⟩\langle \vec{S} \rangle = \langle \Psi | \hat{\vec{S}} |\Psi\rangle ⟨S⟩=⟨Ψ∣S^∣Ψ⟩

L'énergie moyenne selon l'axe OxOxOx :

⟨Sx⟩=ℏ2(α∗β∗)(0110)(αβ)=ℏ2(α∗β+β∗α)\langle S^x \rangle = \frac{\hbar}{2} \begin{pmatrix} \alpha^* & \beta^* \end{pmatrix} \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix} \begin{pmatrix} \alpha \\ \beta \end{pmatrix} = \frac{\hbar}{2} \left ( \alpha^*\beta+\beta^*\alpha \right )⟨Sx⟩=2ℏ​(α∗​β∗​)(01​10​)(αβ​)=2ℏ​(α∗β+β∗α)

L'énergie moyenne selon l'axe OyOyOy :

⟨Sy⟩=ℏ2(α∗β∗)(0i−i0)(αβ)=ℏ2i(α∗β−β∗α)\langle S^y \rangle = \frac{\hbar}{2} \begin{pmatrix} \alpha^* & \beta^* \end{pmatrix} \begin{pmatrix} 0 & i \\ -i & 0 \end{pmatrix} \begin{pmatrix} \alpha \\ \beta \end{pmatrix} = \frac{\hbar}{2} i\left ( \alpha^*\beta-\beta^*\alpha \right )⟨Sy⟩=2ℏ​(α∗​β∗​)(0−i​i0​)(αβ​)=2ℏ​i(α∗β−β∗α)

L'énergie moyenne selon l'axe OzOzOz :

⟨Sz⟩=ℏ2(α∗β∗)(100−1)(αβ)=ℏ2(∣α∣2−∣β∣2)\langle S^z \rangle = \frac{\hbar}{2} \begin{pmatrix} \alpha^* & \beta^* \end{pmatrix} \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix} \begin{pmatrix} \alpha \\ \beta \end{pmatrix} = \frac{\hbar}{2} \left ( |\alpha|^2-|\beta|^2 \right )⟨Sz⟩=2ℏ​(α∗​β∗​)(10​0−1​)(αβ​)=2ℏ​(∣α∣2−∣β∣2)

Nous pouvons constater quelque chose d'intéressant : si nous avons des probabilités non nulles d'avoir un spin vers le haut ou vers le bas selon l'axe OzOzOz, alors nous avons une énergie moyenne non nulle dans les autres axes de mesure OxOxOx et OyOyOy et donc des probabilités non nulles d'avoir des états selon ces axes.

(⟨Sx⟩⟨Sy⟩⟨Sz⟩)=ℏ2(α∗β+αβ∗i(α∗β−αβ∗)∣α∣2−∣β∣2)\begin{pmatrix} \langle S^x \rangle \\ \langle S^y \rangle \\ \langle S^z \rangle \end{pmatrix} = \frac{\hbar}{2} \begin{pmatrix} \alpha^*\beta+\alpha\beta^* \\ i\left ( \alpha^*\beta-\alpha\beta^* \right ) \\ |\alpha|^2-|\beta|^2 \end{pmatrix}​⟨Sx⟩⟨Sy⟩⟨Sz⟩​​=2ℏ​​α∗β+αβ∗i(α∗β−αβ∗)∣α∣2−∣β∣2​​

Précession de Larmor​

Pour le reste de notre étude nous allons supposer que nous n'avons que ce terme dans l'hamiltonien :

H=−qℏ2mσ⃗⋅B⃗\mathcal{H} = - \frac{q\hbar}{2m} \vec{\sigma}\cdot\vec{B} H=−2mqℏ​σ⋅B

Nous avons cette équation :

iℏ∂Φ∂t=−qℏ2mσ⃗⋅B⃗Φi\hbar \frac{\partial\Phi}{\partial t} = - \frac{q\hbar}{2m}\vec{\sigma}\cdot\vec{B} \Phiiℏ∂t∂Φ​=−2mqℏ​σ⋅BΦ

La solution de cette équation est de cette forme :

Φ(t)=exp⁡(iq2mσ⃗⋅B⃗t)Φ0\Phi(t) = \exp{\left ( i\frac{q}{2m} \vec{\sigma}\cdot\vec{B} t \right )}\Phi_0 Φ(t)=exp(i2mq​σ⋅Bt)Φ0​

Mais une rotation selon un axe θ⃗\vec{\theta}θ pour un spineur est la suivante :

R(θ⃗)=exp⁡(i12σ⃗⋅θ⃗t)R(\vec{\theta}) = \exp{\left ( i\frac{1}{2}\vec{\sigma}\cdot\vec{\theta} t\right )}R(θ)=exp(i21​σ⋅θt)

Nous avons alors l'axe de rotation :

θ⃗=qmB⃗\vec{\theta} = \frac{q}{m} \vec{B}θ=mq​B

Il s'agit de la précession de Larmor. Nous avons donc aussi une pulsation associée :

ωL≡qBm\omega_L \equiv \frac{qB}{m}ωL​≡mqB​

Celle-ci ne dépend que de l'intensité du champ magnétique appliqué, la masse de l'objet et sa charge.
