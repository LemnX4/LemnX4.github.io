Observables

Définition​

Une observable est une quantité physique mesurable. Mathématiquement une observables est un opérateur qui possèdes des valeurs propres réelles. Soit A^\hat{A}A^ une observable et ∣Ψ⟩| \Psi \rangle∣Ψ⟩ un état propre de valeur propre aaa réelle :

A^∣Ψ⟩=a∣Ψ⟩\hat{A}| \Psi \rangle = a| \Psi \rangleA^∣Ψ⟩=a∣Ψ⟩

Si on fait le produit scalaire avec l'état propre :

⟨Ψ∣A^∣Ψ⟩=a⟨Ψ∣Ψ⟩\langle \Psi |\hat{A}| \Psi \rangle = a \langle \Psi | \Psi \rangle⟨Ψ∣A^∣Ψ⟩=a⟨Ψ∣Ψ⟩

Et maintenant effectuons la transconjuguaisons. Le membre de droite de l'équation reste inchangée parce qu'elle est réelle :

(⟨Ψ∣A^∣Ψ⟩)†=(∣Ψ⟩)†A^†(⟨Ψ∣)†=⟨Ψ∣A^†∣Ψ⟩\left ( \langle \Psi |\hat{A}| \Psi \rangle \right )^\dag = \left ( | \Psi \rangle \right )^\dag \hat{A}^\dag \left ( \langle \Psi |\right )^\dag = \langle \Psi | \hat{A}^\dag | \Psi \rangle(⟨Ψ∣A^∣Ψ⟩)†=(∣Ψ⟩)†A^†(⟨Ψ∣)†=⟨Ψ∣A^†∣Ψ⟩

Nous avons alors cette égalité à respecter :

⟨Ψ∣A^∣Ψ⟩=⟨Ψ∣A^†∣Ψ⟩\langle \Psi |\hat{A}| \Psi \rangle = \langle \Psi | \hat{A}^\dag | \Psi \rangle⟨Ψ∣A^∣Ψ⟩=⟨Ψ∣A^†∣Ψ⟩

Cela nous donne une condition d'hermiticité d'un opérateur pour qu'il soit considéré comme une observable.

Exemple de manipulation d'une observable​

Soit A^\hat{A}A^ un opérateur définis dans la base B(∣1⟩,∣2⟩)B(|1\rangle, |2\rangle)B(∣1⟩,∣2⟩) de cette façon :

A^∣1⟩=−2i∣2⟩\hat{A}|1\rangle = -2i|2\rangleA^∣1⟩=−2i∣2⟩ A^∣2⟩=2i∣1⟩\hat{A}|2\rangle = 2i|1\rangleA^∣2⟩=2i∣1⟩

Il est alors possible d'en déduire la représentation matricielle de A^\hat{A}A^ :

A^=(02i−2i0)B\hat{A} = \begin{pmatrix} 0 & 2i \\ -2i & 0 \end{pmatrix}_BA^=(0−2i​2i0​)B​

En effet, par exemple :

A^∣1⟩=(02i−2i0)(10)=(0−2i)=−2i∣2⟩\hat{A}|1\rangle = \begin{pmatrix} 0 & 2i \\ -2i & 0 \end{pmatrix} \begin{pmatrix} 1 \\ 0 \end{pmatrix} = \begin{pmatrix} 0 \\ -2i \end{pmatrix} = -2i|2\rangleA^∣1⟩=(0−2i​2i0​)(10​)=(0−2i​)=−2i∣2⟩

On peut remarquer que A^=A^†\hat{A}=\hat{A}^\dagA^=A^† : il s'agit donc d'une observable. Il est possible de déterminer les valeurs propres de celle-ci :

A^∣Ψ⟩=λ∣Ψ⟩\hat{A}|\Psi\rangle = \lambda|\Psi\rangleA^∣Ψ⟩=λ∣Ψ⟩

Il faut alors résoudre cette équation :

∣−λ2i−2i−λ∣=0\begin{vmatrix} -\lambda & 2i \\ -2i & -\lambda \end{vmatrix} = 0​−λ−2i​2i−λ​​=0

L'équation caractéristique qui en découle est la suivante :

λ2−4=0\lambda^2 -4 = 0λ2−4=0

D'où les valeurs propres λ1=2\lambda_1=2λ1​=2 et λ2=−2\lambda_2=-2λ2​=−2. On peut déterminer les vecteurs propres. Pour λ=λ1=2\lambda=\lambda_1=2λ=λ1​=2 :

(−22i−2i−2)(ab)=(00)\begin{pmatrix} -2 & 2i \\ -2i & -2 \end{pmatrix} \begin{pmatrix} a \\ b \end{pmatrix} = \begin{pmatrix} 0 \\ 0 \end{pmatrix}(−2−2i​2i−2​)(ab​)=(00​)

Cette équation est possible si −2a+2ib=0-2a+2ib=0−2a+2ib=0 et −2ia−2b=0-2ia-2b=0−2ia−2b=0. Cela reviens à dire que a=iba=iba=ib.

∣Ψ1⟩=(ibb)|\Psi_1\rangle = \begin{pmatrix} ib \\ b \end{pmatrix}∣Ψ1​⟩=(ibb​)

Pour que ce vecteur propre soit un état quantique, il faut qu'il soit unitaire. Nous avons alors ce premier vecteur propre :

∣Ψ1⟩=12(i1)B|\Psi_1\rangle = \frac{1}{\sqrt{2}} \begin{pmatrix} i \\ 1 \end{pmatrix}_B∣Ψ1​⟩=2​1​(i1​)B​

Pour λ=λ2=−2\lambda=\lambda_2=-2λ=λ2​=−2 :

(22i−2i2)(ab)=(00)\begin{pmatrix} 2 & 2i \\ -2i & 2 \end{pmatrix} \begin{pmatrix} a \\ b \end{pmatrix} = \begin{pmatrix} 0 \\ 0 \end{pmatrix}(2−2i​2i2​)(ab​)=(00​)

Cette équation est possible si 2a+2ib=02a+2ib=02a+2ib=0 et −2ia+2b=0-2ia+2b=0−2ia+2b=0. Cela reviens à dire que a=−iba=-iba=−ib.

Nous avons alors ce second vecteur propre :

∣Ψ2⟩=12(−i1)B|\Psi_2\rangle = \frac{1}{\sqrt{2}} \begin{pmatrix} -i \\ 1 \end{pmatrix}_B∣Ψ2​⟩=2​1​(−i1​)B​

Il est tout à fait possible de reprendre les vecteurs de la base pour réécrire les expressions des vecteurs propres :

∣Ψ1⟩=i2∣1⟩+12∣2⟩|\Psi_1\rangle = \frac{i}{\sqrt{2}}|1\rangle + \frac{1}{\sqrt{2}}|2\rangle∣Ψ1​⟩=2​i​∣1⟩+2​1​∣2⟩ ∣Ψ2⟩=−i2∣1⟩+12∣2⟩|\Psi_2\rangle = -\frac{i}{\sqrt{2}}|1\rangle + \frac{1}{\sqrt{2}}|2\rangle∣Ψ2​⟩=−2​i​∣1⟩+2​1​∣2⟩

Ces états sont les états quantiques où il est possible de mesurer quelque chose en appliquant l'observable A^\hat{A}A^.

Si on applique A^\hat{A}A^ à l'état ∣Ψ1⟩|\Psi_1\rangle∣Ψ1​⟩, on obtient la valeur propre 222 et si on l'applique à l'état ∣Ψ2⟩|\Psi_2\rangle∣Ψ2​⟩, on obtient la valeur propre −2-2−2.

Valeur moyenne​

Expression​

La valeur moyenne ⟨A^⟩\langle \hat{A} \rangle⟨A^⟩ des valeurs possibles des observables de l'opérateur A^\hat{A}A^ sur l'état ∣Ψ⟩| \Psi \rangle∣Ψ⟩ est donnée par cette expression :

⟨A^⟩=⟨Ψ∣A^∣Ψ⟩⟨Ψ∣Ψ⟩\langle \hat{A} \rangle = \frac{\langle \Psi | \hat{A}| \Psi \rangle}{\langle \Psi | \Psi \rangle}⟨A^⟩=⟨Ψ∣Ψ⟩⟨Ψ∣A^∣Ψ⟩​

On peut prendre l'exemple de la quantité de mouvement p⃗^=−iℏ∇⃗\hat{\vec{p}} = -i\hbar\vec{\nabla}p​^​=−iℏ∇ où on suppose la fonction d'onde normalisée :

⟨p⃗^⟩=−iℏ⟨Ψ∣∇⃗∣Ψ⟩\langle \hat{\vec{p}} \rangle = -i\hbar\langle \Psi | \vec{\nabla} | \Psi \rangle⟨p​^​⟩=−iℏ⟨Ψ∣∇∣Ψ⟩

Il s'agit alors du produit scalaire entre ∣Ψ⟩| \Psi \rangle∣Ψ⟩ et ∇⃗∣Ψ⟩\vec{\nabla} | \Psi \rangle∇∣Ψ⟩ :

⟨p⃗^⟩=−iℏ∭R3Ψ∗(t,r⃗)∇⃗Ψ(r⃗,t)d3r\langle \hat{\vec{p}} \rangle = -i\hbar\iiint_{\mathbb{R}^3} \Psi^*(t,\vec{r}) \vec{\nabla} \Psi(\vec{r},t) d^3r⟨p​^​⟩=−iℏ∭R3​Ψ∗(t,r)∇Ψ(r,t)d3r

Construction​

La valeur moyenne d'une observable possèdant plusieurs vecteurs propres ∣en⟩| e_n \rangle∣en​⟩ avec les valeurs propres ana_nan​ associées peut se calculer de cette façon :

⟨A^⟩≡∑nanP(an)\langle \hat{A} \rangle \equiv \sum_n a_n \mathbb{P}(a_n)⟨A^⟩≡n∑​an​P(an​)

On connaît la probabilité que la quantité mesurée soit ana_nan​ dans la base des vecteurs propres dans le cas général où il n'y pas eu normalisation l'état :

P(an)=∣⟨en∣Ψ⟩∣2⟨Ψ∣Ψ⟩\mathbb{P}(a_n) = \frac{|\langle e_n | \Psi \rangle|^2}{\langle \Psi | \Psi \rangle}P(an​)=⟨Ψ∣Ψ⟩∣⟨en​∣Ψ⟩∣2​

Il est possible d'expliciter le conjugué et donc aussi de faire commuter le produit scalaire :

P(an)=⟨Ψ∣en⟩⟨en∣Ψ⟩⟨Ψ∣Ψ⟩\mathbb{P}(a_n) = \frac{\langle \Psi | e_n \rangle\langle e_n | \Psi \rangle}{\langle \Psi | \Psi \rangle}P(an​)=⟨Ψ∣Ψ⟩⟨Ψ∣en​⟩⟨en​∣Ψ⟩​

La valeur moyenne peut alors s'écrire de cette façon : :

⟨A^⟩=∑nan⟨Ψ∣en⟩⟨en∣Ψ⟩⟨Ψ∣Ψ⟩=1⟨Ψ∣Ψ⟩⟨Ψ∣(∑nan∣en⟩⟨en∣)∣Ψ⟩\langle \hat{A} \rangle = \sum_n a_n \frac{\langle \Psi | e_n \rangle\langle e_n | \Psi \rangle}{\langle \Psi | \Psi \rangle} = \frac{1}{\langle \Psi | \Psi \rangle} \langle \Psi |\left ( \sum_n a_n | e_n \rangle\langle e_n | \right ) | \Psi \rangle⟨A^⟩=n∑​an​⟨Ψ∣Ψ⟩⟨Ψ∣en​⟩⟨en​∣Ψ⟩​=⟨Ψ∣Ψ⟩1​⟨Ψ∣(n∑​an​∣en​⟩⟨en​∣)∣Ψ⟩

Puisque ∣en⟩| e_n \rangle∣en​⟩ sont les vecteurs propres, il est possible d'écrire an∣en⟩=A^∣en⟩a_n| e_n \rangle = \hat{A}| e_n \ranglean​∣en​⟩=A^∣en​⟩ :

⟨Ψ∣(∑nan∣en⟩⟨en∣)∣Ψ⟩=⟨Ψ∣(∑nA^∣en⟩⟨en∣)∣Ψ⟩\langle \Psi |\left ( \sum_n a_n | e_n \rangle\langle e_n | \right ) | \Psi \rangle = \langle \Psi |\left ( \sum_n \hat{A} | e_n \rangle\langle e_n | \right ) | \Psi \rangle⟨Ψ∣(n∑​an​∣en​⟩⟨en​∣)∣Ψ⟩=⟨Ψ∣(n∑​A^∣en​⟩⟨en​∣)∣Ψ⟩

On a alors cette expression :

⟨A^⟩=1⟨Ψ∣Ψ⟩⟨Ψ∣(∑n∣en⟩⟨en∣)A^∣Ψ⟩\langle \hat{A} \rangle = \frac{1}{\langle \Psi | \Psi \rangle}\langle \Psi |\left ( \sum_n | e_n \rangle\langle e_n | \right ) \hat{A}| \Psi \rangle⟨A^⟩=⟨Ψ∣Ψ⟩1​⟨Ψ∣(n∑​∣en​⟩⟨en​∣)A^∣Ψ⟩

D'après la relation de fermeture nous avons alors l'expression de la valeur moyenne d'une observable.
