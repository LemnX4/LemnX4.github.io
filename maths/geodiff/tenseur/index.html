Définition géométrique​

Définition par transformations​

Les coordonnées sont contravariantes par définition : pour que la position reste la même, les coordonnées doivent changer de manière inverse à la façon dont les vecteurs de la base changent. Nous avons alors la relation suivante pour une transformation de quadrivecteur contravariant :

V′i=∂x′i∂xjVjV'^i = \frac{\partial x'^i}{\partial x^j} V^jV′i=∂xj∂x′i​Vj

Pour un quadrivecteur covariant, il s'agit de l'inverse :

Vi′=∂xj∂x′iVjV'_i = \frac{\partial x^j}{\partial x'^i} V_jVi′​=∂x′i∂xj​Vj​

On peut donc exprimer les anciens quadrivecteurs en fonction des nouveaux :

Vi=∂xi∂x′jV′jV^i = \frac{\partial x^i}{\partial x'^j} V'^jVi=∂x′j∂xi​V′j Vi=∂x′j∂xiVj′V_i = \frac{\partial x'^j}{\partial x^i}V'_jVi​=∂xi∂x′j​Vj′​

Un tenseur est une généralisation des vecteurs à multiples indices contravariants ou covariants. Ils répondent aux mêmes transformations que les quadrivecteurs, il y a juste plusieurs indices et donc plusieurs transformations à prendre en compte. \ Par exemple, si on multiplie tensoriellement deux quadrivecteurs NiN^iNi et PjP^jPj on obtient un tenseur XijX^{ij}Xij :

Xij=NiPjX^{ij} = N^i P^jXij=NiPj

On souhaite maintenant faire un changement de coordonnées :

X′ij=N′iP′j=(∂x′i∂xaNa)(∂x′j∂xbPb)=∂x′i∂xa∂x′j∂xbNaPbX'^{ij} = N'^i P'^j = \left ( \frac{\partial x'^i}{\partial x^a} N^{a} \right ) \left ( \frac{\partial x'^j}{\partial x^b} P^{b} \right ) = \frac{\partial x'^i}{\partial x^a}\frac{\partial x'^j}{\partial x^b}N^a P^bX′ij=N′iP′j=(∂xa∂x′i​Na)(∂xb∂x′j​Pb)=∂xa∂x′i​∂xb∂x′j​NaPb

Nous obtenons cette loi de transformation pour le tenseur :

X′ij=∂x′i∂xa∂x′j∂xbXabX'^{ij} = \frac{\partial x'^i}{\partial x^a}\frac{\partial x'^j}{\partial x^b}X^{ab}X′ij=∂xa∂x′i​∂xb∂x′j​Xab

On voit immédiatement que nous pouvons construire un nombre infini de tenseurs possèdants autant d'indices que nous voulons. De plus, les transformations sont assez simples puisqu'elles sont le produit de celles de quadrivecteurs. Nous montrerons aussi que ce genre d'égalité est vraie pour tout les tenseurs de cette forme puisqu'une égalité entre tenseurs est vraie pour tout systèmes de coordonnées. \ Pour généraliser, un tenseur TTT possédant nnn indices iii_iii​ contravariants et mmm indices jjj_jjj​ covariants s'écrira de cette façon :

T         {j0...jm}{i0...in}T^{\{ i_0 ... i_n\}}_{\ \ \ \ \ \ \ \ \ \{ j_0... j_m \}}T         {j0​...jm​}{i0​...in​}​

Par exemple, on peut écrire un tenseur 2 fois contravariant et 3 fois covariant comme ceci :

T   cijabT^{ab}_{\ \ \ cij}T   cijab​

Ordre​

Un tenseur d'ordre kkk possède nnn indices contravariants et mmm indices covariants tel que k=n+mk=n+mk=n+m avec nnn et mmm des entiers naturels. Par exemple, T   cijabT^{ab}_{\ \ \ cij}T   cijab​ est un tenseur d'ordre 5.

Un tenseur d'ordre 0 est un scalaire.

Un tenseur d'ordre 1 est un vecteur.

On peut toujours, de manière assez évidente et intuitive, multiplier des tenseurs pour en obtenir un nouveau :

Multiplier des tenseurs donne toujours un nouveau tenseur, mais un tenseur n'est pas nécessairement décomposable en un produit d'autres.

Par exemple :

T ija=AaBijT^{a}_{\ ij} = A^a B_{ij}T ija​=AaBij​

On peut vérifier cette propriété en regardant comment ces tenseurs se transforment. Étant donné qu'il s'agit de facteurs multiplicatifs, les deux cotés de l'égalité se transformerons de la même façon.

Importance des tenseurs en physique

Ces objets mathématiques sont très importants en physique puisque nous avons cette affirmation :

Une égalité entre tenseurs est vraie dans tout les référentiels.

Il est facile de vérifier cela : prenons par exemple une égalité entre le tenseur AabA_{ab}Aab​ et le tenseur BabB_{ab}Bab​ multiplié par une constante kkk, un scalaire, dans un référentiel quelconque :

Aab=kBabA_{ab} = k B_{ab}Aab​=kBab​

Or, d'après les lois de transformations des tenseurs, nous pouvons exprimer ces tenseurs dans un autre système de coordonnées :

∂x′i∂xa∂x′j∂xbAij′=k∂x′i∂xa∂x′j∂xbBij′\frac{\partial x'^i}{\partial x^a} \frac{\partial x'^j}{\partial x^b}A'_{ij} = k\frac{\partial x'^i}{\partial x^a} \frac{\partial x'^j}{\partial x^b} B'_{ij}∂xa∂x′i​∂xb∂x′j​Aij′​=k∂xa∂x′i​∂xb∂x′j​Bij′​

Nous pouvons simplifier et on obtient cette équation :

Aij′=kBij′A'_{ij} = k B'_{ij}Aij′​=kBij′​

Il s'agit de la même égalité que la première, mais dans un autre référentiel. Cette affirmation en implique d'autres :

Un scalaire est un réel dont sa valeur ne dépend pas du référentiel choisis.

Si un tenseur est nul ou possède des propriétés de symétries ou de transformations dans un référentiel choisis, alors il possède les mêmes propriétés dans tout les autres référentiels.
