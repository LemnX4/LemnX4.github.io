Tenseur de Riemann

Le tenseur de Riemann est une quantité permettant d'obtenir des informations sur la courbure locale d'une variété riemannienne.

Définition​

Le tenseur de Riemann R bijaR^a_{ \ bij}R bija​ est définit ainsi :

∇i(∇jAa)−∇j(∇iAa)=R bijaAb\nabla_i \left ( \nabla_j A^a \right ) - \nabla_j \left ( \nabla_i A^a \right ) = R^a_{ \ bij} A^b∇i​(∇j​Aa)−∇j​(∇i​Aa)=R bija​Ab

Son expression explicite est la suivante :

R bija=∂iΓ jba−∂jΓ iba+Γ icaΓ jbc−Γ jcaΓ ibcR^a_{ \ bij} = \partial_i \Gamma^a_{\ j b} - \partial_j \Gamma^a_{\ i b} + \Gamma^a_{\ ic} \Gamma^c_{\ jb} - \Gamma^a_{\ jc} \Gamma^c_{\ ib}R bija​=∂i​Γ jba​−∂j​Γ iba​+Γ ica​Γ jbc​−Γ jca​Γ ibc​

Ce tenseur possède des propriétés de symétries appellées identités de Bianchi :

R bija+R jbia+R ijba=0R^{a}_{\ bij} + R^{a}_{\ jbi} + R^{a}_{\ ijb} =0R bija​+R jbia​+R ijba​=0 ∇kR bija+∇iR bjka+∇jR bkia=0\nabla_k R^{a}_{\ bij} + \nabla_i R^{a}_{\ bjk} + \nabla_j R^{a}_{\ bki} = 0∇k​R bija​+∇i​R bjka​+∇j​R bkia​=0

En abaissant le premier indice on peut définir RabijR_{abij}Rabij​ la forme totalement covariante de ce tenseur exprimée en fonction du tenseur métrique et des symboles de Christoffel :

Rabij=12(∂a∂jgbi+∂b∂igaj−∂a∂igbj−∂b∂jgai)−gck{Γ aicΓ bjk−Γ ajcΓ bik}R_{abij} = \frac{1}{2} \left ( \partial_a\partial_j g_{bi} + \partial_b\partial_i g_{aj} - \partial_a\partial_i g_{bj} - \partial_b\partial_j g_{ai} \right ) -g_{ck}\left \{ \Gamma^c_{\ ai}\Gamma^k_{\ bj} - \Gamma^c_{\ aj}\Gamma^k_{\ bi} \right \}Rabij​=21​(∂a​∂j​gbi​+∂b​∂i​gaj​−∂a​∂i​gbj​−∂b​∂j​gai​)−gck​{Γ aic​Γ bjk​−Γ ajc​Γ bik​}

∇i(∇jAa)−∇j(∇iAa)=(∂iΓ jba−∂jΓ iba+Γ icaΓ jbc−Γ jcaΓ ibc)Ab\nabla_i \left ( \nabla_j A^a \right ) - \nabla_j \left ( \nabla_i A^a \right ) = \left ( \partial_i \Gamma^a_{\ j b} - \partial_j \Gamma^a_{\ i b} + \Gamma^a_{\ i c} \Gamma^c_{\ j b} - \Gamma^a_{\ j c} \Gamma^c_{\ i b} \right ) A^b∇i​(∇j​Aa)−∇j​(∇i​Aa)=(∂i​Γ jba​−∂j​Γ iba​+Γ ica​Γ jbc​−Γ jca​Γ ibc​)Ab

On obtient le tenseur de Riemann.

Construction par transports parallèles​

Afin d'obtenir une information précise sur la courbure d'une variété, le transport parallèle est un très bon candidat. Par exemple sur Terre, si vous décidez de partir de l'équateur en regardant toujours dans la même direction, de remonter jusqu'au pôle Nord, de redescendre jusqu'à un autre endroit de l'équateur, puis enfin revenir sur votre point de départ, vous constateriez qu'il y a eu un changement d'angle : la direction que vous allez regarder n'est plus la même. Il faut donc faire une boucle et regarder quelle est la différence entre le vecteur initial et ce qu'on obtient aprés les transports successifs de celui-ci pour avoir une information sur la courbure.

La façon de faire mathématiquement un transport parallèle est d'utiliser la dérivée covariante. En effet, on voudrait que la variation d'un vecteur sur la variété soit nulle ce qui implique donc que la dérivée covariante du ce vecteur est nulle. Mathématiquement :

∇iAa=0\nabla_i A^a = 0∇i​Aa=0

Ce qui nous donne explicitement :

∂iAa=−AbΓ bia\partial_i A^a = -A^b \Gamma^a_{\ bi}∂i​Aa=−AbΓ bia​

Construction des identités de Bianchi​

Pour facilité certains calculs, nous nous placerons dans le référentiel évoqué précédemment. Rappelons d'abord l'expression du tenseur de Riemann covariant :

Rabij=12(∂a∂jgbi+∂b∂igaj−∂a∂igbj−∂b∂jgai)R_{abij} = \frac{1}{2} \left ( \partial_a\partial_j g_{bi} + \partial_b\partial_i g_{aj} - \partial_a\partial_i g_{bj} - \partial_b\partial_j g_{ai} \right )Rabij​=21​(∂a​∂j​gbi​+∂b​∂i​gaj​−∂a​∂i​gbj​−∂b​∂j​gai​) −gck{Γ aicΓ bjk−Γ ajcΓ bik}-g_{ck}\left \{ \Gamma^c_{\ ai}\Gamma^k_{\ bj} - \Gamma^c_{\ aj}\Gamma^k_{\ bi} \right \}−gck​{Γ aic​Γ bjk​−Γ ajc​Γ bik​}

Plaçons nous maintenant dans une géométrie euclidienne. Cela implique directement que les symboles de Christoffel sont nuls et on a donc une nouvelle expression pour le tenseur de Riemann covariant, une expression optimisée pour traiter de ses propriétés de symétries :

Rabij=12(∂a∂jgbi+∂b∂igaj−∂a∂igbj−∂b∂jgai)R_{abij} = \frac{1}{2} \left ( \partial_a\partial_j g_{bi} + \partial_b\partial_i g_{aj} - \partial_a\partial_i g_{bj} - \partial_b\partial_j g_{ai} \right )Rabij​=21​(∂a​∂j​gbi​+∂b​∂i​gaj​−∂a​∂i​gbj​−∂b​∂j​gai​)

Les égalités entre tenseurs étant valables dans tout les référentiels, nous pouvons utiliser cette forme pour obtenir des égalités plus faciles à déterminer grâce à cette nouvelle forme.

Calculons l'égalité suivante par permutation circulaire b→j→i→bb \rightarrow j \rightarrow i \rightarrow bb→j→i→b :

Rabij+Rajbi+RaijbR_{abij} + R_{ajbi} + R_{aijb} Rabij​+Rajbi​+Raijb​ =12(∂a∂jgbi+∂b∂igaj−∂a∂igbj−∂b∂jgai)= \frac{1}{2} \left ( \partial_a\partial_j g_{bi} + \partial_b\partial_i g_{aj} - \partial_a\partial_i g_{bj} - \partial_b\partial_j g_{ai} \right )=21​(∂a​∂j​gbi​+∂b​∂i​gaj​−∂a​∂i​gbj​−∂b​∂j​gai​) +12(∂a∂igjb+∂j∂bgai−∂a∂bgji−∂j∂igab)+ \frac{1}{2} \left ( \partial_a\partial_i g_{jb} + \partial_j\partial_b g_{ai} - \partial_a\partial_b g_{ji} - \partial_j\partial_i g_{ab} \right )+21​(∂a​∂i​gjb​+∂j​∂b​gai​−∂a​∂b​gji​−∂j​∂i​gab​) +12(∂a∂bgij+∂i∂jgab−∂a∂jgib−∂i∂bgaj)+ \frac{1}{2} \left ( \partial_a\partial_b g_{ij} + \partial_i\partial_j g_{ab} - \partial_a\partial_j g_{ib} - \partial_i\partial_b g_{aj} \right )+21​(∂a​∂b​gij​+∂i​∂j​gab​−∂a​∂j​gib​−∂i​∂b​gaj​) =0=0=0

Nous retrouvons la première identité de Bianchi.

∂kRabij+∂iRabjk+∂jRabki\partial_k R_{abij} + \partial_i R_{abjk} + \partial_j R_{abki}∂k​Rabij​+∂i​Rabjk​+∂j​Rabki​ =12(∂k∂a∂jgbi+∂k∂b∂igaj−∂k∂a∂igbj−∂k∂b∂jgai)= \frac{1}{2} \left ( \partial_k\partial_a\partial_j g_{bi} + \partial_k\partial_b\partial_i g_{aj} - \partial_k\partial_a\partial_i g_{bj} - \partial_k\partial_b\partial_j g_{ai} \right )=21​(∂k​∂a​∂j​gbi​+∂k​∂b​∂i​gaj​−∂k​∂a​∂i​gbj​−∂k​∂b​∂j​gai​) +12(∂i∂a∂kgbj+∂i∂b∂jgak−∂i∂a∂jgbk−∂i∂b∂kgaj)+ \frac{1}{2} \left ( \partial_i\partial_a\partial_k g_{bj} + \partial_i\partial_b\partial_j g_{ak} - \partial_i\partial_a\partial_j g_{bk} - \partial_i\partial_b\partial_k g_{aj} \right )+21​(∂i​∂a​∂k​gbj​+∂i​∂b​∂j​gak​−∂i​∂a​∂j​gbk​−∂i​∂b​∂k​gaj​) +12(∂j∂a∂igbk+∂j∂b∂kgai−∂j∂a∂kgbi−∂j∂b∂igak)+ \frac{1}{2} \left ( \partial_j\partial_a\partial_i g_{bk} + \partial_j\partial_b\partial_k g_{ai} - \partial_j\partial_a\partial_k g_{bi} - \partial_j\partial_b\partial_i g_{ak} \right )+21​(∂j​∂a​∂i​gbk​+∂j​∂b​∂k​gai​−∂j​∂a​∂k​gbi​−∂j​∂b​∂i​gak​) =0=0=0
