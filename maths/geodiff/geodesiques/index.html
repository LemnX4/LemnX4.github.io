Équation des géodésiques

L'équation des géodésique permet de déterminer une courbe de plus court chemin entre deux points d'un espace à géométrie non euclidienne.

Définition​

L'équation qui détermine quelles sont les variations des coordonnées xix^ixi en fonction d'un paramètre affine sss pour décrire une courbe de plus court chemin entre deux points est la suivante :

d2xads2+Γ ijadxidsdxjds=0\frac{d^2x^a}{ds^2} + \Gamma^{a}_{\ ij} \frac{dx^i}{ds}\frac{dx^j}{ds} = 0ds2d2xa​+Γ ija​dsdxi​dsdxj​=0

Une courbe qui vérifie cette équation est appellée géodésique. Le paramètre sss est dit affine car la transformation affine s′=as+bs'=as+bs′=as+b laisse invariant cette équation.

Démonstration​

Puisque ξi\xi^iξi sont les coordonnées carthésiennes en géométrie euclidienne le chemin le plus court est une droite. On peut mathématiquement décrire une droite de cette façon :

d2ξbds2=0\frac{d^2\xi^b}{ds^2} = 0ds2d2ξb​=0

Pour passer aux coordonnées générales on utilise la dérivée en chaîne :

dξbds=dxids∂ξb∂xi\frac{d\xi^b}{ds} = \frac{dx^i}{ds}\frac{\partial \xi^b}{\partial x^i}dsdξb​=dsdxi​∂xi∂ξb​

L'équation de la droite est alors la suivante :

dds(dxids∂ξb∂xi)=0\frac{d}{ds} \left ( \frac{dx^i}{ds}\frac{\partial \xi^b}{\partial x^i} \right ) = 0dsd​(dsdxi​∂xi∂ξb​)=0

On peut développer la dérivée :

dds(dxids∂ξb∂xi)=dds(dxids)∂ξb∂xi+dxidsdds(∂ξb∂xi)=d2xids2∂ξb∂xi+dxidsdds(∂ξb∂xi)\frac{d}{ds} \left ( \frac{dx^i}{ds}\frac{\partial \xi^b}{ \partial x^i} \right ) = \frac{d}{ds} \left ( \frac{dx^i}{ds} \right )\frac{\partial \xi^b}{\partial x^i} + \frac{dx^i}{ds} \frac{d}{ds} \left ( \frac{\partial \xi^b}{\partial x^i} \right ) = \frac{d^2x^i}{ds^2}\frac{\partial \xi^b}{ \partial x^i} + \frac{dx^i}{ds} \frac{d}{ds} \left ( \frac{\partial \xi^b}{\partial x^i} \right )dsd​(dsdxi​∂xi∂ξb​)=dsd​(dsdxi​)∂xi∂ξb​+dsdxi​dsd​(∂xi∂ξb​)=ds2d2xi​∂xi∂ξb​+dsdxi​dsd​(∂xi∂ξb​)

Nous allons utiliser l'identité dds=dxjds∂∂xj\frac{d}{ds} = \frac{dx^j}{ds} \frac{\partial}{\partial x^j}dsd​=dsdxj​∂xj∂​ sur le terme de droite :

dxidsdds(∂ξb∂xi)=dxidsdxjds∂∂xj(∂ξb∂xi)=dxidsdxjds∂2ξb∂xi∂xj\frac{dx^i}{ds} \frac{d}{ds} \left ( \frac{\partial \xi^b}{\partial x^i} \right ) = \frac{dx^i}{ds} \frac{dx^j}{ds} \frac{\partial}{\partial x^j} \left ( \frac{\partial \xi^b}{\partial x^i} \right ) = \frac{dx^i}{ds} \frac{dx^j}{ds} \frac{\partial^2 \xi^b}{\partial x^i\partial x^j}dsdxi​dsd​(∂xi∂ξb​)=dsdxi​dsdxj​∂xj∂​(∂xi∂ξb​)=dsdxi​dsdxj​∂xi∂xj∂2ξb​

Injectons ces résultats dans l'équation de la droite :

d2xids2∂ξb∂xi+∂2ξb∂xi∂xjdxidsdxjds=0\frac{d^2x^i}{ds^2}\frac{\partial \xi^b}{\partial x^i} + \frac{\partial^2 \xi^b}{\partial x^i\partial x^j}\frac{dx^i}{ds} \frac{dx^j}{ds} = 0ds2d2xi​∂xi∂ξb​+∂xi∂xj∂2ξb​dsdxi​dsdxj​=0

On isole le terme d2xids2\frac{d^2x^i}{ds^2}ds2d2xi​ en multipliant l'équation par la quantité ∂xa∂ξb\frac{\partial x^a}{\partial \xi^b}∂ξb∂xa​ :

d2xids2∂xa∂ξb∂ξb∂xi+∂xa∂ξb∂2ξb∂xi∂xjdxidsdxjds=d2xids2δia+∂xa∂ξb∂2ξb∂xi∂xjdxidsdxjds=d2xads2+∂xa∂ξb∂2ξb∂xi∂xjdxidsdxjds\frac{d^2x^i}{ds^2}\frac{\partial x^a}{\partial \xi^b}\frac{\partial \xi^b}{\partial x^i} + \frac{\partial x^a}{\partial \xi^b}\frac{\partial^2 \xi^b}{\partial x^i\partial x^j}\frac{dx^i}{ds} \frac{dx^j}{ds} = \frac{d^2x^i}{ds^2}\delta_i^a + \frac{\partial x^a}{\partial \xi^b}\frac{\partial^2 \xi^b}{\partial x^i\partial x^j}\frac{dx^i}{ds} \frac{dx^j}{ds} = \frac{d^2x^a}{ds^2} + \frac{\partial x^a}{\partial \xi^b}\frac{\partial^2 \xi^b}{\partial x^i\partial x^j}\frac{dx^i}{ds} \frac{dx^j}{ds}ds2d2xi​∂ξb∂xa​∂xi∂ξb​+∂ξb∂xa​∂xi∂xj∂2ξb​dsdxi​dsdxj​=ds2d2xi​δia​+∂ξb∂xa​∂xi∂xj∂2ξb​dsdxi​dsdxj​=ds2d2xa​+∂ξb∂xa​∂xi∂xj∂2ξb​dsdxi​dsdxj​

On retrouve le symbole de Christoffel Γ ija\Gamma^{a}_{\ ij}Γ ija​ ce qui nous donne l'équation des géodésiques.
