Arrangement

L'arrangement est une opération permettant de déterminer le normbre maximum d'arrangements possibles d'objets discernables dans des boîtes.

Définition

Soit AnkA_n^kAnk​ l'arrangements possibles de kkk objets discernables dans nnn emplacements avec k≤nk\leq nk≤n. Il y a alors des emplacements contenants des objets mais aussi des emplacements vides. Cette quantité est définie comme le nombre de façons de disposer kkk objets discernables dans nnn emplacements. Cette quantité correspond aussi au nombre de permutations entre les objets mais aussi les permutations entre les objets et un emplacement vide. Si on prend toutes les permutations possibles, nous avons trois types de permutations : Celles entre les objets, celles entre les objets et un emplacement vides et celles entre les emplacements vides. Le nombre total de permutations est n!n!n!, le nombre de permutations entre les emplacements vides est (n−k)!(n-k)!(n−k)! et le reste est le nombre de permutations AnkA_n^kAnk​. On a alors cette relation :

n!=Ank×(n−k)!n! = A_n^k \times (n-k)!n!=Ank​×(n−k)!

Les permutations se multiplient puisque si on a une permutation sur les emplacement vide, il faut de nouveau prendre en compte toutes les autres permutations entre les objets et entre les objets et un emplacement vide. \ On peut alors isoler l'arrangement :

Ank=n!(n−k)!A_n^k = \frac{n!}{(n-k)!}Ank​=(n−k)!n!​
