Combinaison

La combinaison est une opération permettant de déterminer le normbre maximum d'arrangements possibles d'objets indiscernables dans boîtes.

Définition​

Soit CnkC_n^kCnk​ la combinaison de kkk objets indiscernables dans nnn boîtes. On dit que cette quantité est kkk parmi nnn. Cette quantité est définie comme le nombre de façons de disposer kkk objets indiscernables dans nnn emplacements. La différence entre l'arrangement est que nous utilisons ici des objets indiscernables. Cela signifie qu'une permutation d'objets laisse invariant le système étudié. L'arrangement AnkA_n^kAnk​ prend en compte le nombre de façons de disposer les objets CnkC_n^kCnk​ dans les emplacements mais aussi le nombre de façons de disposer les objets entre eux, k!k!k!. Nous avons alors cette relation :

Ank=Cnk×k!A_n^k = C_n^k\times k!Ank​=Cnk​×k!

On peut alors isoler la combinaison :

Cnk=n!k!(n−k)!C_n^k = \frac{n!}{k!(n-k)!}Cnk​=k!(n−k)!n!​

Une autre notation existe pour la combinaison :

(nk)≡Cnk\begin{pmatrix} n \\ k\end{pmatrix} \equiv C_n^k(nk​)≡Cnk​

Elle répond à certaines propriétés :

(nn)=1\begin{pmatrix} n \\ n\end{pmatrix} = 1(nn​)=1 (nk)=(nn−k)\begin{pmatrix} n \\ k\end{pmatrix} = \begin{pmatrix} n \\ n-k\end{pmatrix}(nk​)=(nn−k​) (nk)=(n−1k−1)+(n−1k)\begin{pmatrix} n \\ k\end{pmatrix} = \begin{pmatrix} n-1 \\ k-1\end{pmatrix} + \begin{pmatrix} n-1 \\ k\end{pmatrix}(nk​)=(n−1k−1​)+(n−1k​)

La dernière propriété est très utile pour construire le triangle de Pascal.
