Factorielle

La factorielle est une opération permettant de déterminer le normbre maximum d'arrangements possibles d'objets discernables.

Définition​

Soit nnn objets discernables. Nous allons les disposer dans nnn emplacements et compter combien de façons il y a de les disposer. \ Pour le premier objet, il y a nnn façons différente de le mettre dans nos emplacements. Une fois le premier objet utilisé, il nous reste n−1n-1n−1 emplacements : il y a alors n−1n-1n−1 façons différentes de mettre le second objet dans les emplacements. Nous pouvons continuer comme ça jusqu'à ce qu'il n'y ai plus qu'un seul emplacements à remplir. Le nombre total de façons de disposer les nnn objets dans les nnn emplacements est alors donné par cette quantité :

n×(n−1)×(n−2)×(n−3)×...×1n\times(n-1)\times(n-2)\times(n-3)\times...\times 1n×(n−1)×(n−2)×(n−3)×...×1

Cette opération sur nnn s'appelle la factorielle et se note n!n!n!. Pour n≥0n\geq0n≥0 :

n!≡n(n−1)(n−2)(n−3)...1n! \equiv n(n-1)(n-2)(n-3)...1n!≡n(n−1)(n−2)(n−3)...1

Elle répond à certaines propriétés :

0!=10!=10!=1 n!=n(n−1)!n!=n(n-1)!n!=n(n−1)!

Cette quantité donne le nombre de mélanges possibles de nnn objets discernables mais aussi le nombre de permutations possibles. Si on a 2 objets, on peut faire 2!=22!=22!=2 permutations. Pour 3 objets, on peut faire 3!=63!=63!=6 permutations. Par exemple :
